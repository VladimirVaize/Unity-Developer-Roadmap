# 📝 Задание: Окно для настройки «Волн врагов» (Enemy Waves)

## Цель:
Создать редакторское окно, которое позволит геймдизайнеру быстро создавать, настраивать и тестировать волны врагов без написания кода.

## Требования:
1. Создайте класс `EnemyWaveWindow`, унаследованный от `EditorWindow`.
   - Добавьте пункт меню `Tools/Редактор волн врагов`.
  
2. Используйте EditorGUILayout для создания следующих полей:
   - `Префаб врага` (ObjectField) — какой враг будет спавниться.
   - `Количество врагов` (IntSlider) — от 1 до 20.
   - `Задержка между спавнами` (Slider) — от 0.2 до 5 секунд.
   - `Радиус спавна` (Slider) — от 1 до 20 метров.
   - `Цвет визуализации` (ColorField) — для отображения радиуса в Scene View.
  
3. Добавьте две кнопки:
   - `▶ Спавнить волну` — создаёт указанное количество врагов в случайных позициях в пределах радиуса спавна.
   - `❌ Удалить всех врагов` — удаляет всех врагов на сцене (используйте тег `Enemy` или `FindObjectsOfType`).
  
4. Добавьте визуализацию в Scene View:
   - С помощью `OnSceneGUI()` нарисуйте окружность (WireDisc) радиусом спавна вокруг центра сцены или вокруг выбранного объекта (по желанию).
  
5. Сохраняйте настройки между сессиями через `EditorPrefs` (количество врагов, радиус, задержку, цвет).
6. Добавьте проверку: если префаб не назначен — показывать предупреждение (`HelpBox`) и блокировать кнопку спавна.

### Дополнительно (по желанию):
- Добавьте возможность выбрать тип волны (Enum): `Обычная`, `Быстрая`, `Танк`.
- Добавьте поле `Имя волны` (TextField) и сохраняйте разные наборы параметров в `ScriptableObject`.
