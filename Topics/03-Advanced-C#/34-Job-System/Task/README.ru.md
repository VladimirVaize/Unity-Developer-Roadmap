# 🧩 Практическое задание: Параллельное обновление позиций пуль

## 🎯 Цель задачи
Реализовать систему пуль в Unity, где каждая пуля обновляет свою позицию в отдельном потоке с помощью `IJobParallelFor`, без риска дедлоков и с максимальной производительностью.

---

## 📝 Описание задачи
У вас есть массив из 10 000 пуль, летящих вперёд. Каждая пуля должна:
1. Иметь текущую позицию (Vector3) и направление (Vector3)
2. Каждый кадр перемещаться на `speed * Time.deltaTime`
3. Если пуля выходит за пределы области (например, дальше 1000 единиц), она либо уничтожается, либо телепортируется обратно

Вы не можете использовать цикл for в основном потоке — это вызовет фризы. Вместо этого вы должны применить Job System.

---

## 🧠 Что нужно реализовать
1. Создать структуру задания `BulletUpdateJob`, реализующую `IJobParallelFor`
2. Внутри задания:
   - Принимать `NativeArray<Vector3>` позиций
   - Принимать `NativeArray<Vector3>` направлений
   - Принимать `float deltaTime` и `float speed`
   - В методе `Execute(int i)` обновлять позицию `[i]`
  
3. В MonoBehavior (например, `BulletManager`) каждые Update:
   - Выделять (или переиспользовать) `NativeArray` с `Allocator.TempJob`
   - Заполнять массивы текущими данными
   - Создать и запланировать задание
   - Вызвать `Complete()` и применить результаты обратно к пулям
   - Вызвать `Dispose()` на NativeArray
  
---

## 🔍 Дополнительные условия (необязательно, но будет плюсом)
- Добавить `[BurstCompile]` для максимальной производительности
- Реализовать переработку массивов, чтобы избежать выделения памяти каждый кадр (использовать `Allocator.Persistent` + ручное управление)
- Реализовать проверку границ внутри задания — если пуля улетела слишком далеко, сбросить её в случайную позицию в пределах радиуса

---

## 📤 Ожидаемый результат
- Плавная работа без спайков в Frame Rate
- Стабильные ~60–200 FPS на 10 000 пуль
- Полное отсутствие дедлоков и ошибок многопоточности
- Код должен быть чистым, с комментариями ключевых моментов

---

## 🚀 Подсказка
```csharp
[BurstCompile]
struct BulletUpdateJob : IJobParallelFor {
    public NativeArray<Vector3> positions;
    [ReadOnly] public NativeArray<Vector3> directions;
    public float deltaTime;
    public float speed;
    
    public void Execute(int i) {
        positions[i] += directions[i] * speed * deltaTime;
        // опционально: проверка границ
    }
}
```
