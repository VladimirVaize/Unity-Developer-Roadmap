# 🧪 Практическая задача по UI Canvas

## 📌 Задача: Создать три типа отображения здоровья

Вы работаете над игрой в жанре RPG. Вам нужно реализовать три разных индикатора здоровья (Health Bar) для одного и того же персонажа, используя три режима рендеринга Canvas:

1. **Screen Space – Overlay** — полоска здоровья должна быть всегда в левом верхнем углу экрана (глобальный HUD).
2. **Screen Space – Camera** — полоска здоровья висит над персонажем, но перекрывается 3D-объектом, если тот подходит ближе к камере, чем персонаж.
3. **World Space** — полоска здоровья является 3D-табличкой над головой персонажа, поворачивается вместе с ним и видна только с определённого расстояния.

---

## 🧱 Что нужно сделать:
1. Создайте сцену с:
   - Игроком (куб или капсула)
   - Камерой (Main Camera)
   - 3D-кубом, который может двигаться (например, по клавише `WASD` или просто разместить его между камерой и игроком)
2. Создайте три отдельных Canvas (или три отдельных индикатора) на одном экране:

### ✅ Canvas 1 – Overlay HUD
- Render Mode = `Screen Space – Overlay`
- Разместите полоску здоровья (Image + Text) в левом верхнем углу
- Не зависит от камеры и 3D-объектов

### ✅ Canvas 2 – Screen Space – Camera (с эффектом перекрытия)
- Render Mode = `Screen Space – Camera`
- `Render Camera` = Main Camera
- `Plane Distance` = 1.5 (расстояние от камеры)
- Полоска здоровья висит над игроком (но позиционируется в экранных координатах, привязанных к игроку — используйте скрипт для привязки `Camera.WorldToScreenPoint`)
- Двигайте 3D-куб между камерой и игроком — куб должен перекрывать полоску здоровья, если он ближе к камере

### ✅ Canvas 3 – World Space
- Render Mode = `World Space`
- Размер Canvas: ширина 2, высота 1
- Сделайте этот Canvas дочерним объектом игрока (чтобы двигался вместе с ним)
- Поверните Canvas лицом к камере (или добавьте скрипт, который делает билборд)
- Добавьте фон, полоску здоровья и текст

3. Напишите простой скрипт `HealthBarUpdater`, который:
   - Управляет значением здоровья (по нажатию клавиш +/-)
   - Одновременно обновляет все три индикатора (ширина полоски и текст)

4. (Необязательно, но полезно) Добавьте эффект, когда персонаж поворачивается — Canvas в World Space поворачивается вместе с ним, а два других остаются статичными.

## 🔍 Проверка результата:
После запуска сцены вы должны увидеть:

| Действие | Ожидание |
|----------------------|--------------------------------|
| Изменение здоровья (+/-) | Все три полоски обновляются одновременно |
| Вращение камеры | Overlay и Camera-UI остаются на экране, World Space поворачивается |
| Перемещение 3D-куба между камерой и игроком | Только полоска в режиме Camera частично перекрывается кубом |
| Приближение/удаление камеры | World Space изменяет видимый размер, Overlay и Camera — нет |
