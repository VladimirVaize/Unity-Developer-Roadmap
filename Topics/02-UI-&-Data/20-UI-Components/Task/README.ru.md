# 🧪 Практическая задача: Адаптивная панель главного меню

### Цель:
Создать панель главного меню, которая работает на любом разрешении экрана (от смартфона до широкого монитора) и использует все три изученных компонента: RectTransform, якоря и контейнеры.

---

## 📋 Условие

### Вам нужно реализовать в Unity UI, который содержит:
- Заголовок (текст "Меню") – прижат к верхнему краю, центрирован горизонтально.
- Три кнопки ("Играть", "Настройки", "Выход") – расположены вертикально одна под другой в центре экрана.
- Панель с информацией – появляется при наведении на любую кнопку (всплывающая подсказка с описанием справа от кнопки).
  Эта панель должна быть привязана к правому краю экрана и автоматически менять свою высоту в зависимости от длины текста.

### Требования к реализации:
1. Использовать `Vertical Layout Group` для вертикального расположения кнопок.
2. Настроить якоря заголовка так, чтобы он оставался прижатым к верху и центрировался при изменении ширины.
3. Панель информации должна быть создана как дочерний элемент Canvas, с `RectTransform` привязанным к правому краю (Anchors: right, top + bottom stretch или right + top, но с динамической высотой).
4. Внутри панели информации использовать `Content Size Fitter` (Vertical Fit = Preferred Size) для автоматического изменения высоты.
5. Все кнопки должны иметь одинаковую ширину, растягивающуюся на всю ширину контейнера, но с отступами слева и справа.

---

## 🔧 Ожидаемый результат
- При разрешении 1920x1080 панель меню выглядит пропорционально.
- При разрешении 800x600 кнопки не обрезаются, заголовок остаётся сверху, панель подсказок не выходит за границы экрана.
- При добавлении длинной подсказки (например, "Открывает экран загрузки сохранений и настроек профиля") панель информации увеличивается по высоте, а текст внутри неё переносится.
