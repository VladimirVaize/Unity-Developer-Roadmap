# 🎨 UI Компоненты Unity: RectTransform, Якоря, Layout Group

> [!Note]
> Этот материал посвящён созданию адаптивного пользовательского интерфейса (UI) в Unity.
> В отличие от обычных 3D-объектов, UI-элементы управляются через RectTransform,
> а их положение и размер относительно экрана задаются с помощью якорей (Anchors) и контейнеров (Layout Group).

---

## 📐 RectTransform
### Для чего нужно:
`RectTransform` — это специальный компонент трансформации для UI-элементов (кнопок, панелей, текста, изображений). 
Он добавляет к обычному Transform понятия ширины (Width) , высоты (Height) , якорей и опорной точки (Pivot).

### Как использовать:
- Выделите любой UI-элемент (например, Image) в иерархии. В Инспекторе вы увидите компонент RectTransform.
- Width / Height — задают размер прямоугольника элемента.
- Pos X, Pos Y — позиция относительно якорей или родителя.
- Pivot — точка вращения и масштабирования (0,0 = левый нижний угол, 1,1 = правый верхний).
- Синие стрелки/квадраты в окне Scene View позволяют изменять размер и вращение визуально.

### Пример:
🟢 Вы создаёте кнопку `Button`. В RectTransform ставите `Width = 160`, `Height = 60`. Pivot оставляете (0.5, 0.5) — центр. Теперь кнопка вращается и масштабируется из центра.

## ⚓ Якоря (Anchors)
### Для чего нужно:
Якоря определяют, к какой части родительского элемента (или экрана) привязан UI-элемент. 
Это основа адаптивности под разные разрешения экрана. 
У каждого RectTransform есть 4 якоря (Min X, Max X, Min Y, Max Y), которые можно перемещать.

Как использовать:
- В окне Scene View выберите UI-элемент. В его углах появятся маленькие белые "радиальные" манипуляторы — это якоря.
- Самый быстрый способ: в Инспекторе нажать на квадратный значок Anchor Presets (слева от слова "RectTransform").
  - Растянуть на весь родитель: Shift + клик на нижний правый пресет (стрелки во все стороны).
  - Привязать к левому краю: выбрать пресет с левой привязкой.
  - Привязать к правой верхней точке: пресет в правом верхнем углу.
- Можно вручную вводить значения Min/Max X/Y в Инспекторе.

### Пример:
📱 Кнопка `Pause` должна всегда висеть в правом верхнем углу, независимо от ширины экрана. 
→ Выбираем Anchor Preset → top-right (стрелка вправо-вверх). 
Теперь Pos X/Y отсчитываются от правого верхнего угла родительского элемента (обычно Canvas).

## 🧩 Контейнеры (Layout Group)
### Для чего нужно:
`Horizontal Layout Group`, `Vertical Layout Group` и `Grid Layout Group` автоматически упорядочивают дочерние UI-элементы. 
Они избавляют от ручного выставления позиций и размеров.

### Как использовать:
1. Создайте пустой GameObject (например, `MenuPanel`) и добавьте на него компонент `Horizontal Layout Group`.
2. Добавьте внутрь несколько дочерних элементов (кнопки, иконки).
3. Настройте параметры в Layout Group:
   - Padding — отступы от краёв.
   - Spacing — расстояние между элементами.
   - Child Alignment — выравнивание по вертикали/горизонтали.
   - Child Controls Size — позволяет контейнеру управлять шириной/высотой детей.
   - Child Force Expand — заставляет детей растягиваться, чтобы заполнить свободное место.
  
### Пример:
🎮 Панель инвентаря: нужно 4 ячейки в ряд.
→ Создаёте `InventoryGrid`, добавляете `Grid Layout Group`. 
Ставите `Cell Size = 80x80`, `Spacing = 10`. 
Добавляете 4 дочерних Image — они автоматически выстроятся в сетку с одинаковыми размерами и промежутками.

---

## 🧠 Сочетание всех трёх
Чаще всего вы используете все эти инструменты вместе:
1. Главный Canvas содержит контейнер (Vertical Layout Group) для экрана настроек.
2. Внутри контейнера — кнопки с RectTransform фиксированной высоты.
3. Каждая кнопка заякорена так, чтобы при растягивании контейнера она оставалась нужной ширины.
4. При изменении разрешения экрана всё перестраивается автоматически.
