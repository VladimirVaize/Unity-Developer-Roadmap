# TextMeshPro (TMP): Современная типографика в Unity

TextMeshPro (часто сокращённо TMP) — это мощное решение для рендеринга текста в Unity, которое пришло на смену стандартному компоненту `Text` (Legacy). 
TMP обеспечивает гораздо лучшее качество, гибкость и производительность, используя технологию SDF (Signed Distance Field).

---

## ✨ Преимущества перед стандартным Text (Legacy)
| Характеристика | Стандартный Text (Legacy) | TextMeshPro |
|-----------------------|-------------------|---------------------------|
| Качество при масштабировании | Пикселизация, размытие | Острый, чёткий текст при любом размере |
| Сглаживание | Ограниченное | Превосходное благодаря SDF |
| Тени и обводки | Требуют дополнительных объектов или шейдеров | Встроенные стили (Outline, Shadow, Bevel) |
| Библиотека иконок | Нет | Встроенные спрайты (Emoji, иконки) |
| Rich Text теги | Базовые (bold, italic, color, size) | Расширенные (градиенты, вращение, тени, спрайты) |
| Производительность | Хорошая для простого текста | Отличная для сложного текста (один меш на весь текст) |
| Выравнивание и перенос | Базовое | Продвинутое (гибкое разбиение слов, отступы) |

Главное преимущество: SDF-технология позволяет векторной информации о шрифте жить в текстуре, поэтому текст остаётся чётким и гладким при масштабировании, вращении и даже прыжках камеры.

## 🔤 Шрифты (Font Assets)
TMP не использует системные шрифты напрямую. Вместо этого вы создаёте Font Asset — специальный ресурс Unity, сгенерированный из файла `.ttf` или `.otf`.

### Как создать Font Asset:
1. Импортируйте `.ttf` или `.otf` файл в папку Assets.
2. Правой кнопкой по файлу → `Create` → `TextMeshPro` → `Font Asset`.
3. Настройте параметры (размер текстуры, диапазон символов, padding) и нажмите `Generate Font Atlas`.

### Основные параметры:
- Atlas Resolution — размер текстурной карты (512x512, 1024x1024...)
- Character Set — какие символы включить (ASCII, Unicode, или вручную)
- Padding — отступ между символами в атласе (влияет на резкость обводок)
- Rendering Mode — режим SDF (обычно `SDFAA` или `SDFAA_HINTED`)

Пример использования: Создайте Font Asset для вашего любимого шрифта, и он будет одинаково чётко выглядеть на 4K-экране и на мобильном устройстве.

## 🎨 Материалы (Materials)
Каждый компонент TMP использует материал, который определяет, как текст рендерится. Unity предоставляет несколько встроенных материалов TMP:

| Материал | Эффект |
|--------------------|------------------------------|
| `TMPro Material` | Базовый текст с поддержкой SDF |
| `TMPro Material Outline` | Добавляет контур (обводку) вокруг букв |
| `TMPro Material Glow` | Свечение (glow) |
| `TMPro Material Bevel` | 3D-объёмный эффект |
| `TMPro Material Gradient` | Градиентная заливка |

Пример настройки: Выберите компонент TMP → в поле `Material` выберите `TMPro Material Outline`. 
Затем в Inspector отрегулируйте `Outline Width = 0.2` и `Outline Color = красный` — каждая буква получит аккуратную красную обводку.

## 🎬 Анимация текста
TMP не имеет встроенного готового компонента анимации, но его можно анимировать:
1. Через код: Меняйте свойства `color`, `fontSize`, `vertexPosition` индивидуально для каждого символа (используя `TMP_TextInfo`).
2. Через Vertex Shader: Продвинутый способ — писать собственный шейдер, который анимирует вершины текста (волна, рябь, смещение).
3. Через Rich Text + AnimationCurve: Комбинируя теги и анимацию в скрипте.

### Простой пример в скрипте (C#):
```csharp
// Пульсация размера текста
float pulse = Mathf.Sin(Time.time * 5f) * 0.2f + 1f;
textComponent.fontSize = 36 * pulse;
```

## 🏷️ Rich Text теги (Расширенная разметка)
TMP поддерживает все Legacy-теги + множество уникальных. Теги вставляются прямо в строку текста.

### Основные теги (работают как в Legacy, так и в TMP):

| Тег | Описание | Пример |
|------------|-----------------|-------------------------|
| `<b>...</b>` | Жирный | `<b>Жирный текст</b>` |
| `<i>...</i>` | Курсив | `<i>Курсив</i>` |
| `<size=30>...</size>` | Размер шрифта | `<size=50>Крупно</size> нормально` |
| `<color=red>...</color>` | Цвет (по имени или #RRGGBB) | `<color=#FF0000>Красный</color>` |

### Расширенные теги TMP (уникальные):

| Тег | Описание | Пример |
|-------------|-------------------|--------------------------|
| `<alpha=#AA>` | Прозрачность (в HEX) | `<alpha=#88>Полупрозрачный текст</alpha>` |
| `<cspace=5>` | Межсимвольный интервал | `<cspace=10>Р а з р я д к а</cspace>` |
| `<voffset=10>` | Смещение по вертикали | Нормальный текст `<voffset=15>поднятый</voffset>` | 
| `<rotate=15>` | Поворот символов | `<rotate=10>Наклонный</rotate>` |
| `<pos=100px>` | Абсолютное позиционирование | Текст `<pos=200px>сдвинут вправо` |
| `<sprite index=0>` | Встроенный спрайт (иконка) | Нажмите `<sprite name="icon_play">` чтобы начать |
| `<gradient>` | Градиент (должен быть задан в материале) | `<gradient>Радужный текст</gradient>` |
| `<link="myid">...</link>` | Создаёт кликабельную область | `<link="shop">Купить</link>` (обработка через `OnPointerClick`)

### Практический пример строки с тегами:
```text
Это <b>жирный</b> и <i>курсивный</i> текст.
<size=40>Огромный текст</size> с <color=yellow>жёлтым</color> акцентом.
<alpha=#66>Полупрозрачный</alpha> и <rotate=10>повёрнутый</rotate>.
Нажмите <sprite name="next"> чтобы продолжить.
```

### Создание собственного спрайта для тега `<sprite>`
1. Подготовьте PNG-иконку.
2. Импортируйте в Unity, настройте `Sprite Mode = Multiple` (если нужно).
3. Откройте `Window → TextMeshPro → Sprite Asset`.
4. Создайте новый Sprite Asset и добавьте в него ваши спрайты.
5. В компоненте TMP укажите этот Sprite Asset в поле `Sprite Asset`.
6. Используйте тег: `<sprite name="myIcon">`.
