# 🎬 Создание простых анимаций по свойствам (Animation Window)

> [!Note]
> Анимация в Unity — это изменение свойств объектов во времени.
> Окно Animation Window позволяет создавать покадровую анимацию без написания кода, просто записывая изменения параметров
> (позиция, поворот, масштаб, цвет, прозрачность и даже переменные ваших скриптов).

---

## 🧠 Основные понятия

| Понятие | Описание |
|------------------|-------------------------------------|
| Animation Clip | Файл с анимацией (например, «открытие двери»). Хранит ключевые кадры. |
| Ключевой кадр (Keyframe) | Момент времени, в который задано конкретное значение свойства. |
| Dopesheet (Раскадровка) | Режим отображения ключевых кадров в виде полосок. |
| Curves (Кривые) | Режим, показывающий плавность изменения значений между кадрами. |

---

## ⚙️ Как открыть Animation Window
`Window → Animation → Animation` (или `Ctrl + 6` на Windows, `Cmd + 6` на Mac)

---

## 🛠️ Создание первой анимации (по шагам)

### 1️⃣ Выберите объект
В Hierarchy выберите GameObject (например, куб `Cube`).

### 2️⃣ Откройте Animation Window и создайте клип
Нажмите `Create` в окне анимации. Укажите имя (например, `RotateAnimation`) и сохраните. Unity автоматически добавит на объект компонент `Animator`.

### 3️⃣ Добавьте свойство для анимации
Нажмите `Add Property` → выберите нужное свойство:
- `Transform` → `Position`, `Rotation`, `Scale`
- `Sprite Renderer` → `Color`
- `Light` → `Intensity`
- или любое публичное поле вашего скрипта (`public float speed`)

### 4️⃣ Создайте ключевые кадры
- Переместите красную полосу (воспроизведения) на момент времени (например, `0:00`)
- Нажмите красную кнопку записи 🔴 (или нажмите на поле свойства и измените значение — точка сохранится автоматически)
- Перейдите на другой момент (например, `1:00`)
- Измените значение свойства (куб повернётся, переместится, сменит цвет)

### 5️⃣ Запустите воспроизведение
- В окне анимации нажмите ▶️ Play (просмотр)
- В редакторе нажмите Play (главный запуск) — анимация зациклится, если стоит галка `Loop Time` в настройках клипа.

## 📐 Пример анимации по свойствам

### Пример 1: Прыгающий кубик (смена Y-позиции)
- Свойство: `Transform.Position.y`
- Кадр `0:00` → `y = 0`
- Кадр `:30` → `y = 2`
- Кадр `1:00` → `y = 0`

Результат: куб плавно подпрыгивает и возвращается.

### Пример 2: Затухание спрайта
- Свойство: `Sprite Renderer.Color.a` (альфа-канал)
- Кадр `0:00` → `a = 1` (полностью видим)
- Кадр `2:00` → `a = 0` (полностью прозрачен)

Результат: объект плавно исчезает за 2 секунды.

### Пример 3: Вращение по оси Z
- Свойство: `Transform.Rotation.z` (в градусах)
- Кадр `0:00` → `z = 0`
- Кадр `2:00` → `z = 360`

Результат: полный оборот за 2 секунды.

---

## 🔁 Как сделать анимацию зацикленной
`Ctrl + D` на клипе в Project Window → выберите созданную копию → в Inspector поставьте галочку **Loop Time**. 
Или выберите клип в **Project Window** и в окне Inspector снизу поставьте `Loop Time: true`.

---

## 🧩 Важный нюанс
Анимация перезаписывает значения свойств во время работы. 
Если вы попытаетесь переместить объект скриптом во время анимации — анимация перетянет значение на себя 
(решается через `AnimatorController` и слои/маски, но это уже продвинутый уровень).
