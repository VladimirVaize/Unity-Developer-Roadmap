# 🎭 Анимационные слои (Layers) и маски: Смешивание анимаций в Unity

## 📌 Зачем это нужно?
В реальных играх персонажи редко делают только одно действие одновременно. 
Например, солдат должен бежать и стрелять, персонаж — идти и махать рукой, а босс — дышать и одновременно атаковать. 
Без специального механизма анимации конфликтовали бы: либо полностью проигрывалась анимация бега, либо стрельбы.

Анимационные слои (Layers) и маски (Avatar Masks) позволяют накладывать анимации друг на друга, разделяя тело на независимые части.

---

## 🧠 Анимационные слои (Animation Layers)

### Что это?
Это как слои в Photoshop, но для анимации. Каждый слой управляет определённым набором движений, а Unity смешивает их по правилам, которые вы задаёте.

### Где находятся?
1. Выберите Animator Controller (в окне Project).
2. Откройте Animator Window (`Window → Animation → Animator`).
3. Слева вы увидите вкладку Layers (по умолчанию есть слой `Base Layer`).

### Ключевые параметры слоя

| Параметр | Что делает |
|------------|-------------------------|
| Weight (Вес) | Насколько сильно этот слой влияет на итоговую анимацию (0 = нисколько, 1 = полностью). |
| Mask (Маска) | Какие части тела этот слой контролирует (например, только руки). |
| Blending (Смешивание) | `Override` — заменяет анимацию из нижних слоёв. `Additive` — добавляет движения поверх (например, вздрагивание). |
| Sync (Синхронизация) | Автоматически повторяет параметры и состояния другого слоя (удобно для правой/левой руки). |
| IK Pass | Включает обратную кинематику для этого слоя. |

### Пример использования слоёв
- Слой 0 (Base Layer): ходьба, бег, прыжки — управляет всем телом.
- Слой 1 (UpperBody): стрельба, махание, атака — вес 1, маска только на верхнюю часть тела.
- Слой 2 (IdleSway): лёгкое покачивание в покое — Additive, вес 0.3.

## 🎭 Маски анимации (Avatar Masks)

### Что это?
Маска — это ассет, который говорит Unity: «Какие кости (части тела) разрешено анимировать этому слою, а какие — нет».

### Как создать маску
1. В окне Project: правой кнопкой → `Create → Avatar Mask`.
2. Назовите, например, `UpperBodyMask`.
3. В инспекторе откроется силуэт персонажа:
   - ✅ Зелёные кости — включены (слой может их анимировать)
   - 🔴 Красные кости — выключены (слой их не трогает)
  
4. Для смешивания (бег + стрельба) обычно делают:
   - Маска верхней части тела: включены позвоночник, плечи, руки, голова, выключены ноги и таз.
   - Маска нижней части тела: наоборот.
  
### Применение маски
1. В Animator Window выберите слой.
2. В поле Mask перетащите созданную маску (например, `UpperBodyMask`).
3. Теперь этот слой будет изменять только верхнюю часть тела.

## 🔄 Смешивание анимаций на практике

### Классический пример: бег + стрельба
Цель: персонаж бежит (`Base Layer`), а верхняя часть тела поворачивается и стреляет (`UpperBodyLayer`).

#### Пошаговая настройка:
1. Создайте два слоя:
   - `Base Layer` — анимации бега, ходьбы, прыжка.
   - `UpperBody` — анимации стрельбы, прицеливания, перезарядки.
  
2. Создайте маску `UpperBodyMask`, отключив в ней все кости ног и таза (оставить позвоночник, руки, голову).
3. Назначьте маску слою `UpperBody` (поле Mask).
4. Установите Blending = Override (по умолчанию). Вес = 1.
5. В скрипте управляйте параметрами:
```csharp
animator.SetFloat("Speed", speed);         // для Base Layer
animator.SetBool("IsShooting", isShooting); // для UpperBody
```

6. В Animator Controller сделайте переход в анимацию стрельбы по условию `IsShooting == true` внутри слоя `UpperBody`.

#### Что получится:
- Когда `IsShooting = false` — видно только бег.
- Когда `IsShooting = true` — ноги продолжают бег (из Base Layer), а руки и корпус берут анимацию стрельбы (из UpperBody Layer).

---

## ⚙️ Типы смешивания: Override vs Additive

| Тип | Поведение | Пример использования |
|-----------|------------------------------------|-------------------------|
| Override | Заменяет анимацию нижележащих слоёв на замаскированных костях | Бег + стрельба (руки полностью заменяются) |
| Additive | Добавляет движения поверх существующих | Лёгкое покачивание при дыхании, эффект удара при получении урона |

---

## 🧪 Полезные советы
- 💡 Синхронизация слоёв (Sync) используется, когда у вас два похожих слоя (например, анимация правой и левой руки). Изменения в одном автоматически применяются к другому.
- 💡 Weight можно менять в коде — делать плавное затухание между слоями.
- 💡 Маски экономят время — не нужно рисовать отдельные анимации для каждого сочетания действий.
- 💡 Отладка: в Animator Window включите `Layers → Debug` — увидите текущий вес каждого слоя.
