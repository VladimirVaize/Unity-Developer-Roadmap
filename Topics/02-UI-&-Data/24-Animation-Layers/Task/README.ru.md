# 🎯 Практическое задание: Бег + стрельба через анимационные слои

## 📖 Описание задачи
Вам нужно реализовать персонажа, который может бежать и стрелять одновременно, используя анимационные слои и маску в Unity. 
Верхняя часть тела должна управляться отдельным слоем, чтобы анимация стрельбы не прерывала бег ног.

---

## 🧱 Что дано (стартовые ассеты)
- Один персонаж с Humanoid-ригом (например, `Soldier`).
- Анимации: `Run`, `Shoot_Pistol`, `Idle`.
- Скрипт-заготовка `PlayerController.cs`, который:
  - Перемещает персонажа по оси X/Z (WASD).
  - При зажатой левой кнопке мыши устанавливает `animator.SetBool("IsShooting", true)`.
 
- Animator Controller `SoldierAnimator` с:
  - Слоем `Base Layer`, содержащим `Idle` и `Run` (с переходом по параметру `Speed > 0.1`).
  - Параметром `IsShooting` (bool).
 
---

## ✅ Ваша задача (по шагам)

### Часть 1 — Создание маски (Upper Body Mask)
1. В окне `Project` создайте Avatar Mask с именем `UpperBodyMask`.
2. Откройте маску и отключите (красный цвет) все кости ног и таза. Оставьте только позвоночник, плечи, руки, голову.
3. Сохраните маску.

### Часть 2 — Настройка слоя (Upper Body Layer)
1. В Animator Controller `SoldierAnimator` создайте новый слой с именем `UpperBody`.
2. Установите Weight = 1, Blending = Override.
3. В поле Mask перетащите `UpperBodyMask`.
4. Внутри слоя `UpperBody`:
   - Создайте пустое состояние `Any State`.
   - Добавьте состояние `Shoot`.
   - Сделайте переход из `Any State` → `Shoot` по условию `IsShooting == true`.
   - В настройках перехода снимите галочку `Has Exit Time` (чтобы стрельба начиналась мгновенно).
   - Сделайте переход обратно в `Any State` (или пустое состояние по умолчанию) при `IsShooting == false`.
  
### Часть 3 — Тестирование
1. Запустите сцену.
2. Зажмите W (бег вперёд).
3. Во время бега зажмите левую кнопку мыши.
4. ✅ Ожидаемый результат: персонаж продолжает бежать ногами, но его корпус и руки выполняют анимацию стрельбы. После отпускания стрельбы — руки возвращаются в анимацию бега.

## ⭐ Дополнительное задание (для продвинутых)
- Добавьте Additive-слой для эффекта отдачи оружия (лёгкое сотрясение камеры или рук при выстреле).
- Настройте смешивание веса слоя через код, чтобы переход в стрельбу был плавным:
```csharp
animator.SetLayerWeight(1, Mathf.Lerp(currentWeight, targetWeight, Time.deltaTime * 5f));
```
- Добавьте анимацию `Reload` на том же слое `UpperBody` по другому условию (`IsReloading`).

---

## 🔍 Критерии успеха
- Создана и применена маска `UpperBodyMask`.
- Слой `UpperBody` имеет корректную маску и настройки.
- При беге + стрельбе ноги не останавливаются.
- Анимация стрельбы воспроизводится только при зажатой ЛКМ.
- (Бонус) Плавное затухание слоя или Additive-эффект.
