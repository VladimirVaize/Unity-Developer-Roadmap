# 🎯 Задача: «Интерактивная система звукового окружения»
Вы разрабатываете 3D-игру от первого лица. 
Вам нужно создать систему звуков, которая реагирует на движение игрока и взаимодействие с объектами. 
Игрок ходит по уровню, подбирает предметы и взаимодействует с источниками звука.

## 📝 Требования к системе:
1. Звуки шагов (3D звук)
   - Разные поверхности: трава, бетон, дерево
   - Громкость шагов зависит от скорости игрока
   - Случайный выбор звука из массива для каждой поверхности
   - Правильные настройки `Min Distance` и `Max Distance` (шаги слышны в радиусе 10 метров)
  
2. Звук подбора предмета (2D звук)
   - Одноразовое воспроизведение через `PlayOneShot`
   - Не зависит от позиции игрока (UI-звук)
   - Случайная высота тона (Pitch) для разнообразия
  
3. Стационарные источники звука (3D звук)
   - 3 объекта в сцене: водопад, костёр, колокольчик на ветру
   - Каждый имеет уникальную кривую затухания (Linear, Logarithmic, Custom)
   - При приближении к источнику звук становится громче
  
4. Эффект Доплера
   - Быстро движущийся объект (летающая тарелка) пролетает мимо игрока
   - При пролёте должен быть слышен характерный сдвиг высоты тона
  
5. UI управление звуком
   - Слайдеры для громкости: Master, Music, SFX
   - Кнопка M для отключения всех звуков
   - Кнопка P для паузы всех звуков
  
---

## 🧰 Технические требования:
- Создайте скрипт `PlayerSoundController` для управления звуками игрока
- Создайте скрипт `WorldSoundSource` для стационарных источников
- Создайте скрипт `UIAudioManager` для управления громкостью
- Используйте `AudioMixer` с хотя бы 3 группами (Master, Music, SFX)
- Для звуков шагов определите поверхности через теги или `TerrainData`

---

## 💡 Ожидаемый результат:
- При движении игрока слышны разные шаги в зависимости от поверхности
- При приближении к костру — звук огня становится громче
- При подборе монетки — короткий приятный звук
- Когда тарелка пролетает — слышен сдвиг тона (у-у-у-ууууу)
