# 🎯 Задача: «Адаптивная музыка для игры-ужастика (Horror Game)»
Вы разрабатываете хоррор-игру от первого лица. Музыка должна адаптироваться к уровню страха и действиям игрока. 
Вам нужно реализовать систему адаптивной музыки с использованием вертикального наложения, снапшотов и динамических параметров.

## 📝 Что нужно реализовать:
### Часть 1: Создание структуры музыки
1. Создайте 4 стема (AudioSource) для вертикального наложения:
   - `AtmosphereLayer` — постоянный низкий фон (жужжание, ветер)
   - `TensionLayer` — высокие напряжённые ноты (скрипки)
   - `ThreatLayer` — низкие басы, далёкий барабан (приближение монстра)
   - `PanicLayer` — хаотичный шум, биение сердца (в панике)
  
2. Создайте AudioMixer `HorrorMixer` со снапшотами:
   - `Calm` — нормальное состояние
   - `Tense` — лёгкое напряжение
   - `NearThreat` — угроза близко
   - `Panic` — погоня/паника
  
### Часть 2: Система уровня страха
3. Реализуйте скрипт `FearManager.cs`, который отслеживает:
   - Расстояние до ближайшего врага (0-30 метров)
   - Видит ли игрок врага (true/false)
   - Уровень здоровья игрока (100% → 0%)
   - Количество исследованных предметов (скорость нарастания страха)
  
4. Метод `CalculateFearLevel()` возвращает значение от 0 до 1:
```text
Уровень страха = (1 - (расстояние / 30)) * 0.4 +
                  (видит врага ? 0.3 : 0) +
                  (1 - здоровье) * 0.2 +
                  (исследовано_предметов / макс_предметов) * 0.1
```

### Часть 3: Адаптация музыки к уровню страха
5. В скрипте `AdaptiveHorrorMusic.cs`:
   - `AtmosphereLayer` всегда активен (громкость 0.3-0.5)
   - `TensionLayer` активен при страхе > 0.3 (громкость = страх)
   - `ThreatLayer` активен при страхе > 0.6 (громкость = (страх-0.6)/0.4)
   - `PanicLayer` активен при страхе > 0.8 (громкость = (страх-0.8)/0.2)
  
6. Переключение снапшотов:
   - Страх < 0.3 → `Calm`
   - 0.3 ≤ страх < 0.6 → `Tense`
   - 0.6 ≤ страх < 0.8 → `NearThreat`
   - страх ≥ 0.8 → `Panic`
  
### Часть 4: Игровые события, влияющие на музыку
7. Реализуйте специальные события:
   - `OnPlayerSpotted()` — враг заметил игрока → мгновенно страх = 0.7
   - `OnItemCollected()` — добавляет страх +0.05 до максимума 0.4
   - `OnEscape()` — страх резко падает до 0.1 за 2 секунды
   - `OnDeath()` — проиграть специальный звук через отдельный снапшот `Death`
  
### Часть 5: Дополнительные эффекты
8. При уровне страха > 0.7 включите Lowpass Filter на мастере с частотой среза 5000 - (страх * 4000) Гц
9. При уровне страха > 0.8 добавьте случайные "скримеры" (короткие звуки) через отдельный AudioSource

---

## 🧰 Требования к реализации:
- Используйте минимум 4 слоя и 4 снапшота
- Все изменения громкости должны быть плавными (через `Mathf.Lerp` в Update)
- Снапшоты переключайте с `TransitionTo` и затуханием 0.5-1.5 секунды
- Добавьте отладочный режим (клавиша F1) для визуализации уровня страха и громкости слоёв на UI

---

## 🔍 Проверка работоспособности:
1. Когда игрок далеко от врагов → страх 0.1 → звучит только `AtmosphereLayer`
2. При приближении врага до 10 метров → страх ~0.4 → включается `TensionLayer` + снапшот `Tense`
3. При виде врага вблизи (3 метра) → страх ~0.7 → `ThreatLayer` + снапшот `NearThreat`
4. Если здоровье упало до 20% и враг видит → страх > 0.8 → `PanicLayer` + `Panic` снапшот + Lowpass
5. При успешном побеге → страх падает до 0.1 за 2 секунды
