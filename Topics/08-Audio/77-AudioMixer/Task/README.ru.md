# 🎯 Задача: «Динамическое звуковое окружение для подземелья»
Вы разрабатываете подземелье в RPG игре. 
Вам нужно создать систему динамического звука, которая реагирует на:
1. Разные зоны подземелья (пещера, зал с эхом, комната с водой)
2. Состояния игрока (обычный режим, бой, скрытность)
3. События (открытие сундука, активация ловушки)

## 📝 Что нужно реализовать:
### Часть 1: Создание AudioMixer и групп
1. Создайте AudioMixer с именем `DungeonMixer`
2. Создайте следующую иерархию групп:
   - `Master`
     - `Music`
     - `SFX`
       - `Footsteps`
       - `UI`
       - `Combat`
      
     - `Ambience`
       - `Drip` (капли воды)
       - `Wind` (ветер)
      
     - `Voice`
    
### Часть 2: Эффекты и снапшоты
3. Добавьте эффекты:
   - На группу `Master` добавьте Lowpass Filter
   - На группу `Voice` добавьте Echo
   - На группу `SFX/Combat` добавьте Distortion (для взрывов)
  
4. Создайте снапшоты:
   - `Normal` — стандартные настройки
   - `Underwater` — Lowpass: 600 Hz, громкость Ambience: -5 dB
   - `Cave` — Reverb: 2 сек затухание, Echo на Voice
   - `CombatIntense` — Distortion на SFX/Combat: 0.3, музыка: +3 dB
   - `StealthMode` — Music: -15 dB, Footsteps: -8 dB
  
### Часть 3: Скрипты управления
5. Напишите скрипт `DungeonAudioZone.cs`:
   - На триггерных зонах переключает снапшоты
   - При входе в зону с водой → `Underwater`
   - При входе в пещеру → `Cave`
   - При выходе → `Normal`
  
6. Напишите скрипт `CombatAudioHandler.cs`:
   - Метод `StartCombat()` → переход к `CombatIntense` за 0.3 сек
   - Метод `EndCombat()` → возврат к `Normal` за 1 сек
   - При получении урона → проиграть звук через группу `SFX/Combat` с искажением
  
7. Напишите скрипт `TreasureChest.cs`:
   - При открытии сундука:
     - Временно (на 0.5 сек) переключить снапшот (например, `TreasureGlow`)
     - Проиграть звук с эффектом Reverb (пространственное эхо)
     - Вернуть предыдущий снапшот
    
### Часть 4: Экспонированные параметры
8. Экспонируйте следующие параметры для управления из кода:
   - `MasterVolume` (главная громкость)
   - `MusicVolume`
   - `SFXVolume`
   - `LowpassCutoff`
   - `EchoWetMix`
  
9. Реализуйте UI-слайдеры для изменения громкости музыки и SFX в реальном времени.

---

## 🧰 Требования к реализации:
- Используйте минимум 3 различных снапшота
- Используйте минимум 3 эффекта (Lowpass, Echo, Reverb/Distortion)
- Все переходы снапшотов должны быть плавными (> 0.2 сек)
- Добавьте комментарии к каждому публичному методу
- В `DEVELOPMENT_BUILD` выводите в консоль информацию о переключении снапшотов

---

## 🔍 Проверка работоспособности:
1. Проверьте, что в зоне воды звук становится "приглушённым" (Lowpass)
2. Проверьте, что в пещере голос имеет эхо
3. Проверьте, что при начале боя музыка становится громче, а звуки ударов — искажёнными
4. Проверьте, что при открытии сундука звук "разлетается" с реверберацией
5. Проверьте, что слайдеры громкости работают и сохраняют значения между сценами
