# 🎯 Задача: «Динамический виньеточный эффект для зоны опасности»
Вы разрабатываете RPG игру. Вам нужно создать динамический полноэкранный эффект, 
который активируется при входе игрока в опасную зону (например, лавовый биом). 
Эффект должен представлять собой виньетку с красным свечением и пульсирующей интенсивностью.

## 📝 Что нужно реализовать:
### Часть 1: Создание Fullscreen Shader Graph
1. Создайте Fullscreen Shader Graph с именем `DangerVignette`.
2. Добавьте параметры:
   - `_Intensity` (float, Range 0–1) — сила эффекта
   - `_VignetteSize` (float, Range 0–1) — радиус виньетки
   - `_VignetteColor` (Color) — цвет свечения (красный)
  
3. Реализуйте логику виньетки:
   - Вычислите расстояние от центра экрана (`UV` → `Distance`)
   - Примените `smoothstep` для создания мягкого края
   - Умножьте цвет на `_VignetteColor` и `_Intensity`
   - Смешайте с исходным цветом (BlitSource)
  
4. Создайте Material с этим шейдером.

### Часть 2: Настройка Full Screen Pass Renderer Feature
5. Добавьте Full Screen Pass Renderer Feature в URP Renderer.
6. Установите:
   - Pass Material → `DangerVignette` Material
   - Injection Point → `After Rendering Post Processing`
   - Requirements → `Color`
  
7. Убедитесь, что эффект отображается.

### Часть 3: Управление через C# скрипт
8. Создайте скрипт `DangerZone.cs`:
   - При входе в триггер (зона опасности) плавно увеличивайте `_Intensity` до 1
   - При выходе плавно уменьшайте до 0
   - Добавьте пульсацию (используя `Mathf.Sin(Time.time)`) когда игрок внутри зоны
  
9. Создайте скрипт `VignetteController.cs`:
    - Получает доступ к Material через `RendererFeature`
    - Содержит методы: `SetIntensity(float)`, `SetVignetteSize(float)`
    - Использует `Material.SetFloat()` для изменения параметров в рантайме
  
### Часть 4: Дополнительные визуальные эффекты
10. Добавьте звуковой эффект при входе в зону (сердцебиение).
11. Сделайте так, чтобы размер виньетки уменьшался при приближении к смерти (здоровье < 20%).
12. Добавьте пост-эффект размытия (Blur) через отдельный Render Pass, который активируется только в зоне опасности.

---

## 🧰 Требования к реализации:
- Используйте Full Screen Pass Renderer Feature для основного эффекта.
- Для размытия создайте кастомный ScriptableRendererFeature.
- Все переходы должны быть плавными (используйте `Mathf.Lerp`).
- Добавьте комментарии к каждому публичному методу.
