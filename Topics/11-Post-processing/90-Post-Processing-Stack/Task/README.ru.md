# 🎯 Задача: «Система пост-эффектов для кинематографической RPG»
Вы разрабатываете RPG игру с кинематографическим стилем. Вам нужно создать систему пост-обработки, которая:
1. Адаптируется под разные локации (лес, подземелье, город)
2. Реагирует на состояния игрока (бой, скрытность, диалог)
3. Создаёт кинематографические эффекты (при кат-сценах)

## 📝 Что нужно реализовать:
### Часть 1: Настройка профилей
1. Создайте 4 профиля PostProcessProfile:
   - `Default` — стандартные настройки (Bloom 0.5, небольшой контраст)
   - `Forest` — тёплый, зелёный оттенок, мягкий Bloom
   - `Dungeon` — холодный, высокий контраст, сильный Bloom от факелов
   - `Cinematic` — сильное размытие (DOF), кинематографическая цветокоррекция
  
2. Для каждого профиля настройте:
   - Bloom (Intensity, Threshold, Tint)
   - Color Grading (Temperature, Contrast, Saturation)
   - Vignette (для Dungeon и Cinematic)
  
### Часть 2: Динамическое переключение
3. Создайте скрипт `ZonePostProcessor.cs`:
   - На триггерных зонах переключает профили
   - При входе в лес → `Forest`
   - При входе в подземелье → `Dungeon`
   - При выходе → `Default`
  
4. Создайте скрипт `StatePostProcessor.cs`:
   - При начале боя → увеличить контраст и насыщенность
   - При скрытности → уменьшить яркость, добавить холодный оттенок
   - При диалоге → включить DOF с фокусом на собеседнике
  
### Часть 3: Anti-aliasing
5. Настройте TAA на камере:
   - Jitter Spread: 0.75
   - Stationary Blending: 0.95
   - Motion Blending: 0.85
   - Добавьте возможность переключения на FXAA в настройках
  
### Часть 4: Depth of Field для кат-сцен
6. Создайте скрипт `CinematicDOF.cs`:
   - При активации кат-сцены плавно включает DOF
   - Фокусируется на активном персонаже
   - Использует Aperture: 2.8, Focal Length: 85mm
  
### Часть 5: Динамический Bloom
7. Создайте скрипт `DynamicBloom.cs`:
   - При получении урона → вспышка Bloom (intensity 2.0 на 0.5 сек)
   - При подборе предмета → мягкая вспышка (intensity 1.0 на 0.3 сек)
  
### Часть 6: Локальные Volume
8. Создайте локальные Volume для:
   - Комнаты с сокровищами — тёплый золотой свет, усиленный Bloom
   - Тёмных коридоров — Vignette, холодный оттенок
   - Магических порталов — фиолетовый оттенок, сильное свечение
  
---

## 🧰 Требования к реализации:
- Используйте минимум 4 различных эффекта (Bloom, DOF, Color Grading, Vignette)
- Все переходы должны быть плавными (> 0.5 сек)
- Добавьте комментарии к каждому публичному методу
- В `DEVELOPMENT_BUILD` выводите информацию о переключении профилей

---

## 🔍 Проверка работоспособности:
1. Пройдите через зоны леса и подземелья — проверьте смену цветокоррекции
2. Начните бой — проверьте изменение контраста и насыщенности
3. Активируйте кат-сцену — проверьте DOF
4. Получите урон — проверьте вспышку Bloom
5. Зайдите в комнату с сокровищами — проверьте локальный Volume

Реализуйте систему сохранения настроек:
- Пользователь может настроить интенсивность каждого эффекта
- Настройки сохраняются в `PlayerPrefs`
- При запуске загружаются сохранённые настройки
