# 🎯 Задача: «Локализация игры для 4 языков»
Вы разрабатываете игру "Word Wizard" — образовательную головоломку для детей. 
Игра должна поддерживать 4 языка: English (en), Русский (ru), Deutsch (de), Español (es).

## 📝 Часть 1: Настройка локализации
1. Установите Unity Localization Package
2. Создайте String Table Collection с именем `UI_Texts`
3. Добавьте все 4 языка
4. Добавьте следующие записи:

| Key | en | ru | de | es |
| --- | --- | --- | --- | --- |
| `title` | Word Wizard | Словесный Маг | Wortzauberer | Mago de Palabras |
| `play` | Play | Играть | Spielen | Jugar |
| `settings` | Settings | Настройки | Einstellungen | Ajustes |
| `score` | Score: {0} | Счёт: {0} | Punktestand: {0} | Puntuación: {0} |
| `time` | Time: {0}s | Время: {0}с | Zeit: {0}s | Tiempo: {0}s |
| `hint` | Hint | Подсказка | Hinweis | Pista |
| `victory` | Victory! | Победа! | Sieg! | ¡Victoria! |
| `defeat` | Try again | Попробуй снова | Versuch es nochmal | Inténtalo de nuevo |

## 📝 Часть 2: Работа с текстом и шрифтами
5. Подберите шрифты для каждого языка:
   - English / Spanish: `Roboto` или `Arial`
   - Russian: `Roboto` с поддержкой кириллицы
   - German: `Roboto` (стандартный, с умлаутами)
  
6. Создайте Font Assets для TextMeshPro для каждого шрифта
7. Настройте Fallback Fonts для поддержки специальных символов
8. Напишите скрипт `FontManager.cs`, который:
   - Следит за сменой языка
   - Автоматически меняет шрифт на всех текстовых элементах
   - Поддерживает разные размеры для разных языков
  
## 📝 Часть 3: Адаптация UI
9. Создайте меню настройки языка:
    - 4 кнопки с флагами (🇺🇸, 🇷🇺, 🇩🇪, 🇪🇸)
    - При нажатии → смена языка без перезагрузки
  
10. Реализуйте автоматическое масштабирование текста:
    - Немецкие слова часто длиннее английских
    - Используйте `Content Size Fitter` или кастомное масштабирование
   
11. Адаптируйте макет UI для разных языков:
    - Кнопки должны расширяться под длинный текст
    - Используйте `Horizontal Layout Group` с `Control Child Size`
   
## 📝 Часть 4: Локализация ассетов
12. Создайте Asset Table Collection с именем `Images`
13. Добавьте локализованные изображения:
    - `banner_en.png`, `banner_ru.png`, `banner_de.png`, `banner_es.png`
    - Каждое изображение содержит название игры на своём языке
   
14. Создайте скрипт `LocalizedBanner.cs`, который:
    - Загружает правильный баннер в зависимости от языка
    - Обновляется при смене языка
   
## 📝 Часть 5: Форматы дат и чисел
15. Реализуйте локализованный счётчик времени:
    - Отображение времени в формате: `Time: 45s` (en), `Время: 45с` (ru)
   
16. Реализуйте локализованный счёт:
    - `Score: 1,234` (en), `Счёт: 1 234` (ru), `Punktestand: 1.234` (de)
   
## 📝 Часть 6: Тестирование локализации
17. Создайте сцену `LocalizationTest` со всеми UI-элементами
18. Добавьте переключатель языка в верхнем углу
19. Проверьте:
    - Все тексты отображаются корректно
    - Шрифты меняются правильно
    - Текст помещается в UI-элементы
    - Время и счёт форматируются корректно
   
---

## 🧰 Требования к реализации:
- Используйте Unity Localization Package
- Все тексты должны быть в таблицах (никакого хардкода)
- Смена языка должна работать в рантайме
- Шрифты должны поддерживать все символы языка
- Код должен быть прокомментирован

## 🏆 Дополнительное задание (опционально):
Реализуйте автоматическое определение языка системы:
- При первом запуске определите язык ОС
- Автоматически установите соответствующий язык в игре
- Если язык не поддерживается — используйте английский
```csharp
public class AutoLanguageDetector : MonoBehaviour
{
    void Start()
    {
        string systemLanguage = Application.systemLanguage.ToString();
        string mappedLanguage = MapSystemLanguage(systemLanguage);
        LanguageManager.Instance.SetLanguage(mappedLanguage);
    }
    
    private string MapSystemLanguage(string systemLang)
    {
        // en → en, ru → ru, de → de, es → es
        // fallback → en
        return systemLang switch
        {
            "English" => "en",
            "Russian" => "ru",
            "German" => "de",
            "Spanish" => "es",
            _ => "en"
        };
    }
}
```
