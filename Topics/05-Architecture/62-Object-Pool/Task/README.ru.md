# 🎯 Практическая задача: Система спавна пуль и врагов через Object Pool

## 📋 Описание задачи
Вам нужно создать систему Object Pool для двух типов объектов: пуль и мелких врагов. 
Система должна демонстрировать преимущества пула перед `Instantiate`/`Destroy`, а также корректно сбрасывать состояние объектов при возврате в пул.

---

## 🧱 Структура задачи
### 📁 Часть 1: Пул для пуль
1. Создайте скрипт `BulletPool` (MonoBehaviour), который:
   - Содержит `[SerializeField] private GameObject bulletPrefab`
   - Содержит `[SerializeField] private int poolSize = 30`
   - Использует `Queue<GameObject>` для хранения пуль
   - В `Awake()` заполняет пул префабами в неактивном состоянии
   - Имеет метод `GetBullet(Vector3 position, Quaternion rotation)` — активирует пулю, устанавливает позицию и возвращает её
   - Имеет метод `ReturnBullet(GameObject bullet)` — деактивирует пулю и возвращает в очередь
  
### 📁 Часть 2: Пул для врагов
1. Создайте скрипт `Enemy` с:
   - Полем `public int health = 3`
   - Методом `TakeDamage(int damage)` — уменьшает здоровье, при достижении 0 вызывает возврат в пул
   - Методом `ResetState()` — сбрасывает здоровье до 3, останавливает все корутины, сбрасывает анимации
   - Методом `Initialize(EnemyPool pool, Vector3 position)` — запоминает владельца пула и устанавливает позицию
  
2. Создайте скрипт `EnemyPool` (аналогично BulletPool), но:
   - Возвращает `Enemy` вместо `GameObject`
   - При выдаче врага вызывает `Initialize()`
   - При возврате вызывает `ResetState()`
  
### 📁 Часть 3: Скрипт спавнера
Создайте скрипт `Spawner`, который:
- Содержит ссылки на `BulletPool` и `EnemyPool`
- В `Update()`:
  - По нажатию `Space` — спавнит пулю из пула в позиции спавнера с случайным направлением
  - По нажатию `E` — спавнит врага из пула в случайной позиции вокруг спавнера
 
- Отображает в консоли количество активных объектов (можно через `FindObjectsOfType` для демонстрации разницы с пулом)

### 📁 Часть 4: Демонстрация преимущества пула
Добавьте в `Spawner` кнопку в Inspector через `[ContextMenu]`:
- `[ContextMenu("Spawn 1000 Bullets With Instantiate")]` — создаёт 1000 пуль через `Instantiate` и уничтожает их через `Destroy` через 1 секунду (замерьте время выполнения)
- `[ContextMenu("Spawn 1000 Bullets With Pool")]` — создаёт 1000 пуль через пул и возвращает их (замерьте время выполнения)
- Выведите в консоль время выполнения обоих методов для сравнения

---

## ✅ Критерии выполнения
1. ✅ Пул создаётся один раз в `Awake()` (никаких `Instantiate` во время игры, кроме случаев расширения пула)
2. ✅ При возврате в пул состояние врага полностью сбрасывается (здоровье, позиция, активные эффекты)
3. ✅ Пуля автоматически возвращается в пул через `Invoke` или `Coroutine` через 2 секунды или при столкновении с триггером
4. ✅ Враг возвращается в пул при `health <= 0`
5. ✅ Используйте `[ContextMenu]` для демонстрации сравнения производительности

---

## 🧩 Дополнительное задание (⭐⭐)
Реализуйте расширяемый пул — если очередь пуста, пул автоматически создаёт новый объект (с логом предупреждения) и добавляет его в пул при возврате. 
Также добавьте метод `PreWarm(int additionalCount)` для предварительного расширения пула.
