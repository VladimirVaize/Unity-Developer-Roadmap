# 🎯 Практическая задача: Система DLC через AssetBundle
## 📋 Описание задачи

Вам нужно создать систему загрузки DLC (дополнительного контента) с использованием AssetBundle. 
Представьте, что у вас есть основная игра, а игрок может купить дополнительный пакет: 
новый уровень с уникальными моделями врагов, текстурами и звуками.

---

## 🧱 Структура задачи
### 📁 Часть 1: Создание контента для DLC
1. Создайте сцену `DLCLevel` с простым окружением (платформы, свет, камера)
2. Создайте префаб уникального врага `DLCEnemy.prefab`
3. Создайте текстуру `DLCEnvironment.png` (можно простой цветной квадрат) и материал на её основе
4. Создайте короткий звуковой клип `DLCBackgroundMusic.mp3` (можно использовать любой короткий файл или заглушку)

### 📁 Часть 2: Настройка AssetBundle
1. Назначьте каждому созданному ресурсу имя AssetBundle: `dlc_content`
2. Напишите скрипт редактора `DLCBundleBuilder.cs` в папке `Editor`, который:
   - Собирает бандл `dlc_content` в папку `AssetBundles/` (вне `Assets`)
   - Копирует готовый бандл в `Assets/StreamingAssets/DLC/` (для локального тестирования)
   - Выводит путь до собранного файла в консоль
  
### 📁 Часть 3: Менеджер загрузки DLC
Создайте скрипт `DLCDownloader.cs`:

Требования:
- Проверяет, куплен ли DLC (имитация — через `PlayerPrefs.GetInt("DLC_Owned", 0)`)
- Если DLC куплен:
  - Загружает бандл из `StreamingAssets/DLC/dlc_content`
  - Из бандла загружает сцену `DLCLevel`, префаб врага, материал и аудиоклип
  - Инстанциирует врага в сцене и запускает музыку
 
- Обрабатывает ошибки: если бандл не найден или повреждён — вывести сообщение в консоль
- Имеет метод `UnloadDLC()` для выгрузки бандла из памяти (`bundle.Unload(true)`)

### 📁 Часть 4: Тестовый UI
Создайте простой UI на сцене с кнопками:
- "Check DLC" — вызывает проверку наличия и загрузку DLC
- "Unload DLC" — выгружает бандл и уничтожает все созданные объекты
- "Simulate Purchase" — устанавливает флаг покупки в `PlayerPrefs`

### 📁 Часть 5: Имитация патча (Дополнительно)
Создайте второй вариант DLC с обновлённым контентом:
- Измените текстуру врага (новый цвет)
- Измените скорость врага в префабе
- Соберите новый бандл с тем же именем, но другой версией
- Реализуйте в `DLCDownloader` проверку версии через манифест

---

## ✅ Критерии выполнения
1. Бандл успешно собирается через редакторный скрипт
2. DLC загружается и работает только после "покупки"
3. Все ресурсы из бандла (сцена, префаб, материал, звук) корректно загружаются
4. Выгрузка бандла освобождает память и удаляет созданные объекты
5. При ошибке загрузки (отсутствие файла, повреждение) игра не падает
