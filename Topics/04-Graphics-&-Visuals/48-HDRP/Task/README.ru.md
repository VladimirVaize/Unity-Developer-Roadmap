# 🧪 Практическое задание: Создание фотореалистичной сцены с HDRP

Цель: Научиться настраивать высококачественное освещение, пост-обработку и материалы в High Definition Render Pipeline.

Задача: Создать небольшую сцену ночного города с неоновой вывеской, мокрой дорогой и объёмным туманом.

---

## 🎯 Что нужно сделать (по шагам)
### Шаг 1: Создание HDRP-проекта
- Откройте Unity Hub → `New Project`
- Выберите шаблон `High Definition 3D`
- Назовите проект `HDRP_NightCity`

### Шаг 2: Базовые объекты
1. Удалите из сцены стандартную камеру (`Main Camera`) и направленный свет (`Directional Light`)
2. Создайте:
   - `GameObject` → `Rendering` → `HDRP Camera` (новая камера)
   - `GameObject` → `3D Object` → `Plane` (земля)
   - `GameObject` → `3D Object` → `Cube` (здание)
  
### Шаг 3: Настройка материалов
1. Создайте материал `NeonMaterial`:
   - В окне `Project` → правой кнопкой → `Create` → `Material`
   - В инспекторе выберите шейдер `HDRP/Lit`
   - Установите цвет Albedo → ярко-розовый (RGB: 255, 0, 150)
   - Поднимите Emissive Intensity до `5` (свечение)
   - Включите галку `Emissive`
  
2. Создайте материал `WetAsphalt`:
   - Шейдер `HDRP/Lit`
   - Цвет Albedo → тёмно-серый
   - Smoothness → `0.9` (мокрый блеск)
   - Metallic → `0.8`
  
### Шаг 4: Освещение
1. Добавьте `Area Light`:
   - `GameObject` → `Light` → `Area Light`
   - Поместите над зданием, как неоновую вывеску
   - Цвет: розовый, Intensity: `20`
  
2. Добавьте `Volumetric Fog`:
   - `GameObject` → `Volume` → `Global Volume`
   - В инспекторе → `Add Override` → `Fog` → выберите `Volumetric Fog`
   - Поставьте `Density` = `0.2`
  
### Шаг 5: Пост-обработка (Volume)
1. В том же `Global Volume` добавьте:
   - `Bloom` → Threshold = `0.9`, Intensity = `1.5`
   - `Depth of Field` → включите, Focus Distance = `10`, Aperture = `8`
   - `Color Grading` → Post-exposure = `0.5`, Contrast = `15`
  
### Шаг 6: Неоновая вывеска (Decal Projector)
1. `GameObject` → `Rendering` → `Decal Projector`
2. Поместите на стену здания
3. В инспекторе назначьте ему материал `NeonMaterial`
4. Масштабируйте в форму прямоугольной вывески

### Шаг 7: Тестирование 
- Нажмите Play и оцените результат:
  - Размытый фон (Depth of Field)
  - Свечение от неона (Bloom)
  - Туман, рассеивающий свет
  - Мокрая дорога с отражениями
 
---

## ✅ Критерии успеха 

| Что проверяем | Как проверить |
| --- | --- |
| Установлен HDRP | В настройках проекта `Graphics` выбран `HDRP Asset` |
| Есть объёмный туман | Сцена выглядит задымлённой, свет от неона виден в воздухе |
| Работает Bloom | Яркий розовый неон создаёт ореол вокруг вывески |
| Материал дороги мокрый | На плоскости видны блики и отражения |
| Decal работает | На стене здания висит розовая неоновая вывеска |

---

## 🧠 Дополнительные идеи для развития
- Добавьте Physically Based Sky и настройте время на полночь.
- Включите Screen Space Reflections для отражений в лужах.
- Добавьте Contact Shadows на маленьком объекте (например, мусорный бак).
- Сделайте анимацию цвета неона (через скрипт, меняя Emissive Intensity).
