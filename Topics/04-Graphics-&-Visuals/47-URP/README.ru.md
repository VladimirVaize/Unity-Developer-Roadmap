# 🎨 Universal Render Pipeline (URP): Настройка, рендер-фичи и преимущества

URP — это готовый к использованию «рендер-пайплайн» (конвейер рендеринга) от Unity, 
предназначенный для создания высокопроизводительной графики на мобильных устройствах, консолях, ПК и в вебе. 
Он пришёл на смену старому встроенному рендерингу (Built-in Render Pipeline) и является более гибким и современным.

В этом руководстве разберём: как настроить URP в проекте, что такое Render Features и почему URP лучше старого подхода.

---

## 1️⃣ Настройка URP в проекте

Чтобы начать использовать URP, нужно создать и установить специальный «URP Asset» – файл, 
в котором хранятся все настройки рендеринга (качество теней, антиалиасинг, разрешение и т.д.).

### 🔧 Пошаговая настройка:
1. Создайте URP Asset:
   - В окне `Project` щёлкните правой кнопкой мыши → `Create` → `Rendering` → `Universal Render Pipeline` → `Pipeline Asset` (и опционально `Pipeline Asset (Forward Renderer)`).
   - Назовите его, например, `URP_Settings`.
  
2. Назначьте URP Asset в настройках проекта:
   - `Edit` → `Project Settings` → `Graphics`.
   - В поле `Scriptable Render Pipeline Settings` перетащите созданный `URP_Settings`.
  
3. Обновите материалы (при переходе со старого проекта):
   - `Edit` → `Render Pipeline → Universal Render Pipeline` → `Upgrade Project Materials to URP Materials`.
  
✅ Готово! Теперь ваш проект использует URP. Все новые материалы автоматически будут создаваться с поддержкой URP (Lit Shader вместо Standard).

## 2️⃣ Render Features (Рендер-фичи)
Render Feature — это возможность добавить дополнительный проход рендеринга в конвейер без написания сложного кода. 
Например: эффект обводки (outline), туман, размытие в движении, экранный эффект дождя и т.д.

### 🧩 Как добавить и настроить Render Feature:
1. Выберите Forward Renderer Asset (обычно он находится внутри URP Asset или отдельно).
2. В Inspector найдите секцию `Render Features`.
3. Нажмите `Add Render Feature` → выберите готовую (например, `Render Objects` или `Screen Space Ambient Occlusion`).
4. Настройте параметры:
   - `Event` — когда выполнять (до/после основной геометрии, после постобработки и т.д.).
   - `Filters` — какие объекты затрагивать (по слою, тегу, шейдеру).
   - `Material` — какой материал применить (например, полупрозрачное выделение).
  
### 🌟 Пример использования:
Вы хотите, чтобы враги при получении урона на секунду становились красными без смены основного материала.
- Добавьте `Render Objects` Feature.
- Настройте `Layer Mask = Enemies`.
- Укажите `Material` с красным цветом и `Blending = Additive`.
- Выберите `Event = AfterRenderingOpaques`.

Теперь, активируя эту фичу через скрипт, вы получите временный эффект.

## 3️⃣ Преимущества URP перед Built-in Render Pipeline

| Особенность | Built-in RP | URP |
| --- | --- | --- |
| 🚀 Производительность на мобильных | Средняя | Высокая (оптимизирован) |
| 🎨 Шейдеры | Standard шейдер | Lit / Unlit / Simple Lit (легче) |
| 🌅 Постобработка (Post-processing) | Через отдельный пакет | Встроенная через Volume |
| ✨ Render Features | ❌ Нет | ✅ Есть (легко расширять) |
| 🔆 Освещение (Lighting) | Несколько типов софт-теней | Основные типы + адаптивные |
| 📱 2D Rendering | Ограничен | Отличный встроенный 2D Renderer |
| 🧪 Будущее поддержки | Поддерживается, но не развивается | Основной пайплайн Unity |

---

## 💡 Дополнительные плюсы:
- Простой переход с Built-in (автоапгрейд материалов).
- Универсальные шейдеры — работают одинаково на всех платформах.
- Поддержка URP Shader Graph — создавайте визуальные эффекты без кода.

---

## 🧠 Резюме
- URP — современный, быстрый и гибкий рендер-пайплайн.
- Настройка занимает 2 минуты: создать Asset и назначить в Project Settings.
- Render Features позволяют добавлять эффекты без программирования.
- Преимущества — производительность, постобработка из коробки, будущее Unity.
