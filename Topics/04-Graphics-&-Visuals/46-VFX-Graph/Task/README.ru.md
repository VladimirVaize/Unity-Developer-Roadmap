# 🧪 Практическое задание: Магический след частиц

## 📖 Описание задачи
Вам нужно создать эффект магического следа, который остаётся за движущимся объектом (например, за игроком или летающим кристаллом). 
Требования к эффекту:
- Частицы должны появляться только при движении объекта (не на месте)
- Частицы угасают по времени (прозрачность уменьшается)
- Частицы имеют случайный цвет в диапазоне (синий → фиолетовый → розовый)
- Частицы уменьшаются в размере в течение жизни
- Частицы не летят в стороны, а плавно опускаются вниз (гравитация)
- Используйте VFX Graph (не классическую систему частиц)

---

## 🛠️ Что нужно сделать (пошагово)
1. Создайте VFX Graph с именем `MagicTrail`
2. Настройте Initialize:
   - Спавн частиц: `Rate over Distance` (частицы на пройденное расстояние)
   - Позиция: `Set Position (Plane)` или `Set Position (Mesh)` с привязкой к объекту
   - Время жизни: `Set Lifetime (Random)` — 1.0–2.5 секунды
   - Цвет: `Set Color (Gradient)` — синий → фиолетовый → розовый (рандомный выбор по градиенту)
   - Размер: `Set Size (Random)` — 0.1–0.4
  
3. Настройте Update:
   - `Gravity (слабая)` — частицы медленно падают
   - `Drag` — небольшое сопротивление
   - `Fade` — прозрачность уменьшается к концу жизни
  
4. Настройте Output:
   - Используйте `Quad Output` с материалом, поддерживающим прозрачность (альфа)
   - Включите `Alpha Blending`
  
5. Привяжите эффект к объекту:
   - Добавьте компонент `Visual Effect` на ваш движущийся объект
   - Укажите созданный VFX Graph `MagicTrail`
   - Убедитесь, что в поле `Spawn Rate` используется `Rate over Distance` (не по времени)
  
6. (Дополнительно, ✨) Добавьте параметр `TrailColor` в Blackboard и управляйте цветом следа из скрипта при изменении скорости

---

## ✅ Критерии успеха
- Эффект не спавнит частицы, когда объект стоит на месте
- Частицы плавно исчезают и уменьшаются
- Цвет следа меняется в указанной гамме
- Эффект не падает ниже 60 FPS при ≥1000 частиц (проверить через `Window → Analysis → Profiler`)
- Вы можете объяснить, почему для этой задачи лучше подходит VFX Graph, а не классическая система частиц

---

## 💡 Подсказка
Используйте блок `Set Spawn Rate` → `Rate over Distance` вместо `Constant`. 
Он автоматически вычисляет пройденное расстояние между кадрами и создаёт частицы пропорционально пути. 
Это ключевой момент для эффекта следа!
