# 📘 Задание: Создание системы частиц "Магический взрыв с искрами и туманом"

Вы — разработчик 2D платформера. Нужно реализовать эффект взрыва магической бомбы, который включает:
- Главный выброс искр (однократный)
- Движение частиц в разные стороны
- Столкновение с землёй (2D коллайдеры) и создание малого эффекта дыма при ударе
- Туман (другая система), который остаётся после взрыва
- Оптимизация под мобильное устройство (максимум 200 частиц)

---

## 🧱 Базовые требования:

1. Система A – «Искры»
   - Эмиссия: 1 Burst с 60 частицами при t=0
   - Shape: Circle (радиус 0.5)
   - Speed: от 2 до 6 в случайном направлении
   - Lifetime: 1 – 1.5 сек
   - Color: жёлтый → оранжевый → красный → прозрачный
   - Collision: World с 2D Colliders, Dampen = 0.7, Kill при первом ударе? Нет — только по истечении жизни.
  
2. Подсистема B – «Малый дым» (Sub-Emitter)
   - Привязывается к `A` на событие Collision
   - Каждая частица A при ударе создаёт 1 частицу B
   - B имеет форму медленно парящего серого дыма
   - Lifetime = 0.8 сек, малое количество (чтобы не было 1000 частиц)
  
3. Система C – «Туман после взрыва»
   - Запускается вручную из скрипта через 0.2 секунды после взрыва
   - Loop = true, Rate over Time = 10
   - Shape: Circle, радиус 2, частицы растут и медленно поднимаются
   - Max Particles = 30 (ограничение)
  
4. Оптимизация
   - Убедитесь, что `Max Particles` в A = 80, в B = 40, в C = 30
   - Используйте простой аддитивный материал
   - Отключите коллизии у системы C (она не сталкивается)
  
---

## 📦 Дополнительно (для саморазвития):
- Добавьте вращение частиц (`Rotation over Lifetime`)
- Используйте `Texture Sheet Animation` с 2x2 атласом искр
- Сделайте так, чтобы частицы B имели случайный размер 0.3–0.7

---

## 🧠 Критерии проверки:
- При запуске сцены бомба взрывается по нажатию клавиши (или по таймеру)
- Искры летят, сталкиваются с платформами, оставляют следы дыма
- Туман появляется после и держится 2–3 секунды, а затем исчезает
- FPS не падает ниже 58 на среднем устройстве
