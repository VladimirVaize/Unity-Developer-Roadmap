# 🧪 Практическое задание: Создание «Эффекта сканирования» (Scanning Effect) с помощью Shader Graph

## 🎯 Цель задания
Создать Unlit шейдер, который накладывает на объект движущуюся полосу сканирования (например, как в радаре или научной фантастике). Шейдер должен работать на любой 3D-модели и содержать:
- Основную текстуру (например, металл или панель).
- Полосу сканирования, которая движется вдоль локальной оси Y объекта.
- Свечение (эмиссию) в области полосы.
- Возможность настройки цвета, скорости и ширины полосы через параметры в материале.

---

## 📝 Требования
1. Тип шейдера: Unlit Shader Graph (URP).
2. Параметры (Blackboard):
   - `MainTexture` (Texture 2D) — основная текстура объекта.
   - `ScanColor` (Color) — цвет полосы сканирования.
   - `Speed` (Float) — скорость движения полосы (от -2 до 2).
   - `Width` (Float) — ширина полосы (от 0.1 до 0.5).
   - `EmissionIntensity` (Float) — интенсивность свечения (1–5).
  
3. Логика графа:
   - Используйте мировые или локальные координаты (через узел `Position` в пространстве объекта).
   - Выделите координату Y (канал G или компонент 1 вектора).
   - Создайте полосу, используя `Sine` или `Sawtooth` волну (`Time` + `Multiply` + `Fraction` или `Sawtooth Wave`).
   - Смешайте (`Lerp`) основную текстуру и `ScanColor` с добавлением эмиссии.
  
4. Дополнительно (по желанию):
   - Добавьте параметр `GlowFalloff`, чтобы края полосы были размытыми.
   - Сделайте так, чтобы полоса работала в UV-пространстве (а не в мировых координатах) — эффект наклейки.
  
---

## 🔧 Ожидаемый результат
- В Scene View на объекте (например, сфера или кастомная модель) движется яркая полоса сверху вниз (или снизу вверх).
- Полоса имеет заданный цвет и ширину.
- В области полосы текстура становится ярче (свечение).
- При изменении `Speed`, `Width`, `ScanColor` в инспекторе материала эффект меняется в реальном времени.

---

## 🧩 Подсказки по узлам

| Что нужно сделать | Узлы (названия на английском) |
| --- | --- |
| Получить Y-координату объекта | `Position` (Object Space) → `Split` (выход G) |
| Создать движущуюся позицию полосы | `Time` → `Multiply` (со Speed) → `Fraction` (цикл 0-1) |
| Ширина полосы | `Subtract` (Y_объекта - Y_полосы) → `Abs` → `Step` (порог = Width) |
| Смешивание текстур | `Sample Texture 2D` (MainTexture) → `Lerp` с ScanColor |
| Добавить свечение | `Multiply` (ScanColor × EmissionIntensity) → `Add` к результату |

- Параметры влияют на скорость, ширину и цвет.
