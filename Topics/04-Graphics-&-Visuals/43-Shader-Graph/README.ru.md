# 🎨 Shader Graph: Визуальное создание шейдеров без кода

> [!Note]
> Shader Graph — это мощный визуальный инструмент в Unity, который позволяет создавать шейдеры, соединяя узлы (nodes) в граф, вместо написания кода на HLSL/GLSL.
> Он идеально подходит для художников, дизайнеров и программистов, которые хотят быстро прототипировать материалы с продвинутой графикой.

> [!Important]
> ⚠️ Требование: Shader Graph работает только с Universal Render Pipeline (URP) или High Definition Render Pipeline (HDRP). Встроенный рендер-пайплайн (Built-in) не поддерживается.

---

## 📦 Типы шейдеров в Shader Graph
В Shader Graph вы выбираете шаблон (Master Stack) при создании. Три основных типа:

### 1. Lit Shader (освещаемый) 💡
- Для чего: Объекты, которые взаимодействуют со светом (источники, тени, отражения). Большинство игровых объектов.
- Особенности: Реагирует на направление, цвет и интенсивность света в сцене. Имеет параметры: Albedo (цвет/текстура), Normal Map (рельеф), Metallic (металличность), Smoothness (гладкость), Emission (свечение).
- Пример использования: Персонажи, окружение (камни, деревья), транспорт, оружие.

### 2. Unlit Shader (неосвещаемый) 🌟
- Для чего: Объекты, которые не должны реагировать на свет. Всегда выглядят одинаково независимо от освещения сцены.
- Особенности: Только цвет/текстура, эмиссия, прозрачность. Очень производительный.
- Пример использования: UI-элементы, спрайты (2D), "плоские" стилизованные эффекты, голограммы, экраны с фиксированным цветом.

### 3. VFX Shader (визуальные эффекты) ✨
- Для чего: Специализированные шейдеры для частиц (Particle System) или эффектов, не требующих сложного освещения.
- Особенности: Часто используют прозрачность, аддитивное смешивание (additive), анимацию текстур во времени, dissolve-эффекты.
- Пример использования: Взрывы, огонь, дым, магия, искры, интерфейсные "вспышки".

## 🧠 Базовые узлы (Nodes) для старта
Узлы — это "кирпичики", которые вы соединяете. Вот самые важные:

| Узел | Что делает | Выход |
| --- | --- | --- |
| `Texture 2D` | Позволяет загрузить текстуру | Цвет (RGBA) |
| `Sample Texture 2D` | Читает текстуру с UV-координатами | Цвет, альфа |
| `Multiply` | Умножает два значения (цвета или числа) | Результат умножения |
| `Add` | Складывает значения | Сумма |
| `Lerp` (Linear Interpolate) | Смешивает два значения по третьему (alpha) | Плавный переход |
| `Time` | Выдаёт текущее игровое время | Sine, Cosine, Delta Time |
| `UV` | Координаты текстуры объекта (0-1) | Vector2 (U, V) |
| `Color` | Задаёт постоянный цвет | RGBA |
| `Normal Vector` | Позволяет искажать нормали (для рельефа) | Vector3 |
| `Polar Coordinates` / `Rotate` | Преобразуют UV для круговых эффектов | UV |

## 🛠️ Как создать свой первый шейдер в Shader Graph
### Пошаговый пример: "Пульсирующий цвет" (Unlit)
1. Создайте шейдер: В окне `Project` → правой кнопкой → `Create` → `Shader Graph` → `URP` → `Unlit Shader Graph`. Назовите его `PulseColor`.
2. Откройте редактор: Дважды кликните по файлу `.shadergraph`.
3. Добавьте параметры: В окне `Blackboard` (левая панель) нажмите `+` → `Color` → назовите `BaseColor`. Добавьте `Vector1` (Float) → назовите `Speed`.
4. Постройте граф:
   - Перетащите `BaseColor` и `Speed` в граф.
   - Добавьте узел `Time` → заберите из него `Sine Time`.
   - Умножьте (`Multiply`) `Sine Time` на 0.5 и прибавьте (`Add`) 0.5 (чтобы значение было между 0 и 1).
   - Создайте второй цвет (например, красный) через узел `Color`.
   - Смешайте (`Lerp`) `BaseColor` и красный цвет, используя полученное значение как `T` (alpha).
   - Результат `Lerp` подключите в Master Stack → `Base Color`.
  
5. Сохраните: `Save Asset` (Ctrl+S).
6. Примените: Создайте материал (Material) из этого шейдера (правой кнопкой по шейдеру → `Create` → `Material`).
   Назначьте материал любому объекту в сцене. Цвет будет плавно пульсировать!

## 💡 Советы и лучшие практики

- 🏷️ Именуйте параметры в Blackboard осмысленно (`Speed`, `GlowIntensity`), чтобы их легко было настраивать в материале.
- 🔍 Используйте Preview (клик по узлу → маленькое окошко), чтобы видеть результат работы узла без подключения к финалу.
- ⚡ Unlit быстрее Lit. Для массовых объектов (сотни частиц) предпочитайте Unlit, если не нужно сложное освещение.
- 📂 Организуйте граф: держите узлы аккуратными, группируйте их в `Group` (выделите несколько → правой кнопкой → `Group`).
- 🎥 Визуализируйте UV: подключите UV-координаты прямо в `Base Color` — вы увидите цветовой градиент, полезно для отладки.
