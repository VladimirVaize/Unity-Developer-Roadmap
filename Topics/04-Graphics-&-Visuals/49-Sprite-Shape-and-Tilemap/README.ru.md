# 🎮 Sprite Shape и Tilemap: Создание 2D уровней (платформеров)

В этом материале мы разберём два мощных инструмента Unity для построения 2D уровней: Tilemap 
(для сеточных, пиксельных, чётких уровней) и Sprite Shape (для плавных, органических, изогнутых форм). 
Они идеально подходят для платформеров, RPG, метроидваний и других 2D игр.

---

## 🧱 Tilemap — для уровней «по клеткам»

### Что это такое?
Tilemap — это система, позволяющая рисовать уровень с помощью «плиток» (тайлов), как мозаикой или пиксель-артом. 
Каждый тайл — это маленький спрайт (например, кусок земли, трава, стена). 
Вы работаете в сетке, что даёт идеальный контроль над коллизиями и физикой.

### 📦 Основные компоненты
- Grid — невидимая сетка, на которой всё держится.
- Tilemap — слой с плитками (один слой = одна «ткань» уровня: земля, трава, вода).
- Tile Palette — окно редактора, где вы храните и выбираете плитки.
- Tilemap Collider 2D + Composite Collider 2D — создают физические границы по форме нарисованных тайлов.

### 🛠️ Как использовать (пошагово)
1. Создайте Grid и Tilemap<br>
   `ПКМ в Hierarchy → 2D Object → Tilemap → Rectangular Tilemap`

2. Откройте Tile Palette<br>
   `Window → 2D → Tile Palette`<br>
   Нажмите `Create New Palette`, выберите размер клетки (обычно 16x16 или 32x32 пикселя).

3. Добавьте тайлы<br>
   Перетащите спрайты (разделенные на клетки) в окно `Tile Palette`. Unity спросит, как нарезать их на тайлы — выберите `Grid by Cell Size`.

4. Рисуйте уровень<br>
   Выберите кисть (Brush) в палитре, нажмите и перетаскивайте по сцене, чтобы рисовать плитки.<br>
   Используйте ластик (Eraser Brush) для удаления.

5. Настройте коллизии<br>
   Выберите Tilemap в Hierarchy → `Add Component` → `Tilemap Collider 2D` → затем `Composite Collider 2D`.<br>
   Отметьте `Used by Composite` в `Tilemap Collider 2D`. Физика будет идеально повторять форму тайлов.

### ✅ Для чего подходит Tilemap
- Чёткие платформеры (как Celeste, Super Mario)
- Подземелья (Roguelike)
- Города / дома / комнаты с чёткими границами

## 🎢 Sprite Shape — для плавных и изогнутых уровней
### Что это такое?
Sprite Shape — это инструмент для создания гибких, органических форм: изогнутые платформы, холмы, пещеры, скруглённые стены. 
Вы задаёте контрольные точки, а Unity автоматически заполняет пространство спрайтами, плавно соединяя их.

### 📦 Основные компоненты
- Sprite Shape Profile — настройка: какие спрайты использовать для границ, углов, заливки.
- Sprite Shape Controller — компонент на объекте, который рисует форму по вашим точкам.
- Open / Closed Shape — открытая линия (платформа) или замкнутая форма (остров, пещера).

### 🛠️ Как использовать (пошагово)
1. Создайте профиль Sprite Shape<br>
   `ПКМ в Project → Create → 2D → Sprite Shape Profile → Open Shape` (для платформы) или `Closed Shape` (для острова).

2. Настройте спрайты в профиле
   - Border (граница) — основной спрайт линии.
   - Fill (заливка) — спрайт для внутренности (для Closed Shape).
   - Корнеры (углы) — опционально, для острых поворотов.
  
3. Создайте контроллер в сцене<br>
   `ПКМ в Hierarchy → 2D Object → Sprite Shape → Open Shape`<br>
   Появится пустая изогнутая линия.

4. Рисуйте форму<br>
   В режиме `Edit Spline` (нажмите `Edit Spline` на контроллере в инспекторе) двигайте точки (Ctrl + клик — добавить точку).<br>
   Точки можно переключать между `Linear` (угол) и `Bezier` (плавный изгиб).

5. Добавьте коллизии<br>
   На объекте Sprite Shape добавьте компоненты:<br>
   `Polygon Collider 2D` (для замкнутой формы) или `Edge Collider 2D` (для открытой линии).<br>
   Или используйте `Sprite Shape Collider` — автоматическую физику по форме.

### ✅ Для чего подходит Sprite Shape
- Плавные платформеры (Ori and the Blind Forest)
- Органические пещеры, холмы, облака
- Фон (далёкие горы, изогнутые линии горизонта)

---

## ⚔️ Tilemap + Sprite Shape: комбинируем
#### Ничто не мешает использовать оба инструмента в одной игре:
- Tilemap — для основного уровня: блоки земли, лестницы, стены комнат.
- Sprite Shape — для декораций: корни деревьев, извилистые тропинки, фоновые холмы.

#### Пример платформера:
Пол комнаты — Tilemap (чётки тайлы). Плавный выступ из земли, нависающий над пропастью — Sprite Shape. 
Игрок может ходить по тайлам и по Sprite Shape, если на оба добавлены коллайдеры.

## 🧠 Итог

| Инструмент | Сетка | Форма | Стиль | Лучше для |
| --- | --- | --- | --- | --- |
| Tilemap | Да | Прямоугольная | Пиксельный, чёткий | Стены, полы, подземелья |
| Sprite Shape | Нет | Произвольная/гладкая | Органический, плавный | Холмы, пещеры, платформы-дуги |
