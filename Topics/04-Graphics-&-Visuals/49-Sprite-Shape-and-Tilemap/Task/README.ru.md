# 🎯 Практическое задание: Платформерная пещера
## 📝 Условие
Твоя задача — создать небольшой уровень для 2D платформера, используя Tilemap для основной земли и стен, а также Sprite Shape для органической пещерной арки или плавного холма.

## 🔧 Требования
1. Сцена Unity (2D)
   - Создай новую 2D сцену.
   - Добавь объект `Grid` и один `Tilemap` для земли (основной пол).
  
2. Tilemap часть
   - Нарисуй пол и стены (минимум 10×5 клеток).
   - Добавь `Tilemap Collider 2D` + `Composite Collider 2D` для физики.
   - Используй любые тайлы (можно стандартные из пакета `2D Tilemap Extras` или свои простые спрайты).
  
3. Sprite Shape часть
   - Создай `Sprite Shape Profile` (Closed или Open Shape).
   - Добавь в сцену `Sprite Shape Controller`.
   - Создай плавный изгиб, который нависает над частью Tilemap (как свод пещеры) или плавный холм, на который может запрыгнуть игрок.
   - Настрой коллизию (`Polygon Collider 2D` или `Sprite Shape Collider`).
  
4. Персонаж (для проверки)
   - Добавь простого квадратного игрока с `Rigidbody 2D`, `Box Collider 2D` и скриптом для движения влево/вправо и прыжка.
   - Убедись, что игрок может:
     - Ходить по Tilemap (полу).
     - Запрыгнуть на Sprite Shape (холм/арку).
     - Не проваливаться сквозь физику ни одного из объектов.
    
5. Визуальные отличия
   - Покрась Tilemap в один цвет (или используй спрайты земли).
   - Покрась Sprite Shape в другой цвет или текстуру (например, траву/камень).

---

## ✨ Что оценивается
- Правильная настройка коллизий на обоих типах объектов.
- Органичное соединение Tilemap и Sprite Shape в одном уровне.
- Работоспособность физики (персонаж не падает сквозь платформы).
- Хотя бы один плавный изгиб (Bezier-точка) в Sprite Shape.
