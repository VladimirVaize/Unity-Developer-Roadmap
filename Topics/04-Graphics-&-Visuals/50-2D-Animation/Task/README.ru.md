# 🧪 Практическое задание: 2D анимация персонажа-героя
Цель: Закрепить навыки работы со Sprite Editor, Sprite Atlas и скелетной анимацией (Anima2D) в Unity. 
Вы создадите анимированного персонажа-героя с бегом, прыжком и сменой оружия.

---

## 📥 Исходные материалы (предоставлены)
1. `HeroSpritesheet.png` — спрайт-лист 1024×512, содержит 12 кадров бега (по 85×85 px) и 6 кадров прыжка (по 85×85 px).
2. `HeroParts.png` — разобранный персонаж на части (голова, торс, руки, ноги, меч).
3. `WeaponIcons` — 5 иконок оружия (32×32 px каждая).

---

## 🎯 Задачи
### Этап 1: Нарезка спрайтов (Sprite Editor)
1. Импортируйте `HeroSpritesheet.png` в Unity.
2. В инспекторе измените `Sprite Mode` на `Multiple`.
3. Откройте Sprite Editor → нарежьте спрайты автоматически (`Slice` → `Type: Automatic`).
4. Убедитесь, что получилось 18 спрайтов (12 бег + 6 прыжок).
5. Для каждого спрайта прыжка установите `Pivot` в положение `Bottom` (основание ног).
6. Нажмите `Apply`.

### Этап 2: Создание покадровой анимации
1. Выделите первые 12 спрайтов бега в окне `Project`.
2. Перетащите их на сцену — Unity автоматически создаст `GameObject` с компонентом `Animator` и анимацией `HeroRun`.
3. Аналогично создайте анимацию прыжка из 6 спрайтов (назовите `HeroJump`).
4. В окне Animator создайте переход (transition) из `HeroRun` в `HeroJump` по триггеру `Jump`.
5. Напишите простой скрипт, который по нажатию `Space` вызывает `animator.SetTrigger("Jump")`.

### Этап 3: Оптимизация со Sprite Atlas
1. Создайте Sprite Atlas с именем `HeroAtlas`.
2. Добавьте в `Objects for Packing` папку, где лежат все спрайты героя.
3. Нажмите `Pack Preview` — убедитесь, что все спрайты упакованы в одну текстуру.
4. Запустите сцену и откройте окно Profiler (`Window` → `Analysis` → `Profiler`) → проверьте уменьшение draw calls (по сравнению с отключенным атласом).

### Этап 4: Скелетная анимация (Anima2D)
1. Импортируйте `HeroParts.png` (разобранный персонаж) в Unity.
2. Установите пакет 2D Animation (Anima2D) через Package Manager.
3. Выделите спрайт → Sprite Editor → откройте вкладку `Skinning Editor`.
4. Создайте кости:
   - Корень в тазу.
   - Позвоночник → шея → голова.
   - Левое плечо → левый локоть → левая кисть.
   - Правое плечо → правый локоть → правая кисть (держит меч).
  
5. Привяжите спрайтовые части к костям (используйте `Auto Weights` или вручную).
6. Создайте анимацию `Attack` в окне `Animation` (на 1 секунду):
   - Вращайте правую руку с мечом (от -45° до +90°).
  
7. Вызовите `animator.Play("Attack")` по клику мыши.

### Этап 5: Смена оружия (дополнительно, со звёздочкой ⭐)
1. Импортируйте 5 иконок оружия (`WeaponIcons`).
2. Объедините их в отдельный Sprite Atlas (`WeaponAtlas`).
3. В скрипте героя создайте массив спрайтов оружия.
4. По нажатию клавиш `1`–`5` меняйте спрайт в руке (для Anima2D это будет слот `RightHand`).

---

## ✅ Критерии успеха
- Спрайт-лист нарезан, анимации бега и прыжка работают.
- Переход между анимациями по пробелу (Jump) работает плавно.
- Sprite Atlas уменьшает draw calls (проверено в Profiler).
- Создана скелетная анимация атаки с вращением руки.
- (⭐) Смена оружия работает без перекомпиляции анимации.
