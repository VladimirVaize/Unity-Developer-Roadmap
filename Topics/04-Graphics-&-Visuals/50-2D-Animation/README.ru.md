# 🎞️ 2D анимация в Unity: Sprite Editor, Sprite Atlas, Skeletal Animation

Этот материал посвящён трём ключевым инструментам Unity для создания 2D-анимации: Sprite Editor (нарезка спрайтов), 
Sprite Atlas (оптимизация памяти) и скелетная анимация (Skeletal Animation) с помощью Anima2D или сторонних решений (например, Spine). 
Вы узнаете, для чего нужен каждый инструмент и как его использовать на практике.

---

## 1. 🖌️ Sprite Editor (Редактор спрайтов)
### Для чего нужно:
Sprite Editor позволяет нарезать один большой спрайт-лист (spritesheet) на множество отдельных спрайтов. 
Это необходимо, когда художник даёт вам один PNG-файл, содержащий все кадры анимации персонажа (например, бег, прыжок, атака) или тайлы для тайловой карты.

### Как использовать:
1. Импортируйте спрайт-лист в окно `Project`.
2. Выделите его, в окне Inspector измените `Sprite Mode` с `Single` на `Multiple`.
3. Нажмите кнопку Sprite Editor (откроется отдельное окно).
4. Используйте инструмент Slice → `Type: Automatic` (или `Grid By Cell Size`). Unity сам определит границы кадров.
5. Нажмите `Slice` → `Apply`. Теперь каждый кадр стал отдельным спрайтом (вы увидите их, раскрыв исходный файл в `Project`).
6. Вручную можно корректировать прямоугольники и задавать `Pivot` (точку вращения, например, внизу стоп персонажа).

### Пример:
У вас есть спрайт-лист `HeroRun.png` размером 1024×256, содержащий 8 кадров бега по 128×128 пикселей. 
Вы нарезаете его в Sprite Editor, получаете 8 спрайтов: `HeroRun_0`, `HeroRun_1` … `HeroRun_7`. 
Затем создаёте анимацию бега через окно Animation (перетаскиваете эти спрайты на таймлайн).

## 2. 📦 Sprite Atlas (Атлас спрайтов)
### Для чего нужно:
Sprite Atlas объединяет множество мелких спрайтов в одну большую текстуру (атлас). 
Это снижает количество вызовов отрисовки (draw calls) и ускоряет рендеринг. 
Особенно важно для мобильных игр и UI.

### Как использовать:
1. В окне `Project` → правой кнопкой мыши → `Create` → `Sprite Atlas`.
2. Назовите его (например, `CharacterAtlas`).
3. В инспекторе атласа нажмите `+` в секции `Objects for Packing` и добавьте все нужные спрайты или целые папки
   (Unity автоматически включит все спрайты внутри).
4. Нажмите Pack Preview — увидите, как спрайты упаковались в одну текстуру.
5. Теперь в коде и префабах используйте исходные спрайты как обычно — Unity автоматически подставит правильные координаты из атласа.
6. Для контроля включите `Allow Rotation` / `Tight Packing` для более плотной упаковки.

### Пример:
У вас есть 50 спрайтов интерфейса (кнопки, иконки) по 32×32 пикселя. Без атласа каждая иконка — отдельный draw call. 
Вы создаёте `UIAtlas`, добавляете туда все иконки, нажимаете Pack Preview — Unity создаёт одну текстуру 1024×1024. 
Draw calls сокращаются с 50 до 1-2.

## 3. 🦴 Skeletal Animation (Анимация костей) – Anima2D и сторонние решения
### Для чего нужно:
Скелетная анимация использует иерархию костей (bones), которые деформируют спрайт. 
Вместо сотен кадров (как в спрайт-листе) вы анимируете положение и вращение костей. 
Это даёт плавные переходы, меньше памяти и возможность смены оружия/одежды без перерисовки.

## 🔧 Anima2D (бесплатный пакет от Unity)
### Как использовать:
1. Установите Anima2D через Package Manager (`Window` → `Package Manager` → поиск `2D Animation`).
2. Импортируйте спрайт персонажа (часто разделённый на части: голова, торс, руки, ноги).
3. Выберите спрайт → в окне `Sprite Editor` используйте инструмент `Skinning Editor` (появится после установки пакета).
4. Создайте кости (Bones): инструментом `Create Bone` поставьте точки от корня (таз) до кончиков пальцев.
5. Привяжите спрайтовые части к костям (веса, weight painting) — каждая часть следует за своей костью.
6. Откройте окно `Animation` (`Window` → `Animation` → `Animation`), создайте новую анимацию (например, `Walk`).
7. Включите запись (красная кнопка) → вращайте и перемещайте кости на разных временных метках — создаются ключи анимации.
8. Готово! Анимация воспроизводится через компонент `Animator`.

### Пример:
Персонаж состоит из 10 частей (разрезан в Photoshop). 
В Anima2D вы создаёте кости: таз, позвоночник, шея, левое плечо и т.д. 
Привязываете каждую часть к соответствующей кости. 
Затем создаёте анимацию «прыжок»: на 0-й секунде ноги согнуты, на 10-й — вытянуты. 
Играет плавно, без отдельных спрайтов.

## 🎭 Сторонние решения (Spine)
### Для чего нужно:
Spine (от Esoteric Software) — профессиональный инструмент для скелетной анимации, более мощный, чем Anima2D. 
Позволяет мешать анимации (blending), привязывать скины (скин-переключатели), создавать события на кадрах.

### Как использовать (интеграция с Unity):
1. Купите лицензию Spine и экспортируйте анимацию в формате `.json` / `.atlas`.
2. Импортируйте Spine Runtime for Unity из Asset Store.
3. Поместите экспортированные файлы в папку `Assets`.
4. Перетащите `.json`-файл на сцену — появится игровой объект с компонентом `SkeletonAnimation`.
5. Управляйте анимацией через код: `skeletonAnimation.AnimationName = "run";`
6. Преимущества: физика скинов, анимация по кривым, инверсная кинематика (IK).

### Пример:
В Spine вы создали персонажа с анимациями `idle`, `walk`, `attack`. 
В Unity просто вызываете `skeletonAnimation.AnimationState.SetAnimation(0, "attack", false)`. 
Анимация воспроизводится один раз, затем возвращается в `idle`. Мгновенное смешивание.

## 4. 🔄 Как выбрать инструмент?

| Ситуация | Инструмент |
| --- | --- |
| Простые покадровые анимации (взрыв, бонус) | Sprite Editor + обычная Animation (frame by frame) |
| Много однотипных объектов (зомби, пули) | Sprite Atlas для оптимизации памяти |
| Персонаж с плавными движениями и сменой частей тела | Anima2D (бесплатно) |
| Профессиональный проект со сложной анимацией (IK, скины) | Spine (платно) |
