# 🎯 Практическая задача: Теги, слои и физические взаимодействия

## 📝 Условие задачи
Вы разрабатываете 3D-игру, в которой игрок может стрелять из лазерной пушки по врагам. 
Вам нужно реализовать следующую логику с использованием тегов, слоёв и маски слоёв в Raycast:

### 1. Объекты и их теги/слои:
  - Игрок (Player) — тег `"Player"`, слой `Player`.
  - Враг (Enemy) — тег `"Enemy"`, слой `Enemy`.
  - Лазерный луч (Laser) — тег `"Laser"`, слой `Projectile`.
  - Стена (Wall) — тег `"Wall"`, слой `Environment`.

### 2. Требования:
  - Лазер должен уничтожать врага при попадании.
  - Лазер НЕ должен проходить сквозь стены (столкновение со стеной уничтожает лазер).
  - Лазер НЕ должен взаимодействовать с игроком (проходить сквозь него без вреда и без уничтожения лазера).
  - При выстреле нужно проверить, есть ли стена между игроком и врагом, с помощью Raycast (используя маску слоёв). Если стена закрывает обзор, выстрел не должен достигаь врага.

### 3. Дополнительно (Physics Matrix):
  - Настройте матрицу столкновений так, чтобы:
    - Лазер (слой `Projectile`) сталкивался с врагом (`Enemy`) и стеной (`Environment`).
    - Лазер НЕ сталкивался с игроком (`Player`) и с другими лазерами (`Projectile`).
   
---

## 🛠️ Что нужно сделать (по шагам)

### Шаг 1: Создание слоёв
  - Откройте `Project Settings` → `Tags and Layers`.
  - Создайте слои: `Player`, `Enemy`, `Projectile`, `Environment` (начиная с User Layer 8).

### Шаг 2: Назначение тегов и слоёв объектам
  - Создайте кубы/сферы для игрока, врага, стены.
  - Назначьте каждому объекту соответствующий тег и слой через Inspector.

### Шаг 3: Настройка Physics Matrix
  - `Project Settings` → `Physics` → `Layer Collision Matrix`.
  - Снимите галочки:
    - `Projectile` × `Player` (лазер проходит сквозь игрока)
    - `Projectile` × `Projectile` (лазеры не сталкиваются)
  - Оставьте галочки:
    - `Projectile` × `Enemy`
    - `Projectile` × `Environment`
   
### Шаг 4: Скрипт для лазера (Laser.cs)
  - Храните скорость лазера (`speed`) и время жизни лазера (`lifetime`) в переменных.
  - В `Start` запускайте уничтожение (`Destroy`) лазере через `lifetime` сек.
  - Каждый кадр (в `Update`) задавайте движение через `transform.Translate`
  - В `OnCollisionEnter`:
    - Проверка тега врага (`CompareTag("Enemy")`)
      - Уничтожайте врага при столкновении
    - Лазер уничтожается при любом столкновении (стена, враг)
   
### Шаг 5: Raycast-проверка перед выстрелом (PlayerShoot.cs)
  - Храните префаб лазера (`GameObject laserPrefab`), стартовую позицию выстрела (`Transform firePoint`) и максимальную дистанцию выстрела (`shootDistance = 50f`)
  - В `Update` при нажатии ЛКМ (`Input.GetButtonDown("Fire1")`)
    - Создаём маску: исключаем слой `Player` (игрок), но включаем `Enemy` и `Environment`
    - Выстреливаем лучем `Raycast` вперед (не забудь передать параметр маски в `Raycast`)
      - Если `Raycast` попал в `"Enemy"` (Сделай проверку по тегу - `CompareTag("Enemy")`):
        - Создаём лазер (`Instantiate(laserPrefab, firePoint.position, firePoint.rotation);`)
        - Выводим сообщение `"Выстрел по врагу!"`
      - Если `Raycast` попал в `"Wall"`:
        - Выводим сообщение `"Путь преграждён стеной. Нельзя стрелять."`
      - Если `Raycast` ни во что не попал:
        - Создай лазер
        - Выведи сообщение `"Свободный выстрел"`
       
### Шаг 6: Проверка работы
  - Поместите стену между игроком и врагом. При нажатии кнопки выстрела Raycast должен обнаружить стену и запретить создание лазера.
  - Уберите стену — выстрел должен долетать до врага и уничтожать его.
  - Лазер должен проходить сквозь игрока (нет столкновения), но ударяться в стену или врага.
