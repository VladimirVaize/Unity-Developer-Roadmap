# 🎯 Практическое задание: Баскетбольный симулятор

## 📝 Условие
Вам нужно создать простую сцену, в которой есть:
- 🏀 Мяч (сфера) с компонентом `Rigidbody`
- 🧱 Корзина (пустой объект с коллайдером-триггером в виде кольца)
- 🕹️ Кнопка броска (нажатие клавиши `Space` или клик по UI-кнопке)
- 📦 Неподвижная платформа (движущийся пол, который толкает мяч)

---

## ✅ Что нужно реализовать:

### 1. Бросок мяча в корзину
При нажатии `Space` мяч получает силу вперёд и вверх (`AddForce` с `ForceMode.Impulse`). Подберите значения, чтобы мяч летел по дуге к корзине.

### 2. Гравитация
Мяч должен падать вниз, если не попадает в корзину. Используйте `Use Gravity = true`.

### 3. Кинематическая платформа
Создайте платформу, которая движется влево-вправо (например, `transform.Translate`).<br>
Установите для неё `Rigidbody` с `isKinematic = true`. Мяч, упавший на платформу, должен двигаться вместе с ней и не проваливаться.

### 4. Попадание в корзину
При входе мяча в триггер корзины:
  - Выведите в консоль `"Score!"`
  - Сбросьте мяч в начальную позицию
  - Обнулите его скорость (`velocity = Vector3.zero`)

---

## 🧠 Дополнительное задание (со звёздочкой)

- Добавьте второй мяч с `isKinematic = true`.
  Он не должен реагировать на бросок и гравитацию, но при столкновении с обычным мячом должен его оттолкнуть
  (проверьте, как работает коллизия между кинематическим и обычным Rigidbody).
- Настройте разные `ForceMode` для броска (например, `Force` vs `Impulse`) и пронаблюдайте разницу.

---

## 🔧 Подсказки

- Чтобы получить ссылку на `Rigidbody` в коде:<br>
  `GetComponent<Rigidbody>()` или `public Rigidbody rb;` (перетащить в инспекторе).
- Обнуление скорости:<br>
  `rb.velocity = Vector3.zero;`
- Для движения платформы:<br>
  `transform.Translate(Vector3.right * speed * Time.deltaTime);` и отскок от границ.
