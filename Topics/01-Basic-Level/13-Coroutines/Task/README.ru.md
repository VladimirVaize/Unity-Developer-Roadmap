# 🎯 Практическая задача: Система усилений (Power-up) с визуальными эффектами

## 📝 Описание задачи

Вам нужно реализовать систему временных усилений для игрового персонажа с использованием корутин. Усиление подбирается на сцене, активируется на 5 секунд, даёт игроку ускорение, а затем деактивируется.

---

## ✅ Требования

### 1. Скрипт <a href="../Solution/PowerUp.cs"><code>PowerUp</code></a> (навешивается на объект-усиление на сцене):
  - При столкновении с игроком (`OnTriggerEnter`) усиление должно исчезнуть (визуально и коллайдер) и вызвать метод активации на игроке.
  - После активации усиление не должно быть видно (можно уничтожить или отключить).

### 2. Скрипт <a href="../Solution/PlayerPowerUp.cs"><code>PlayerPowerUp</code></a> (навешивается на игрока):
  - Метод `ActivateSpeedBoost(float duration)` — запускает корутину усиления.
  - Корутина должна:
      - Увеличить скорость игрока (например, `moveSpeed *= 2`).
      - Запустить визуальный эффект (менять цвет спрайта на синий или увеличить масштаб на 20%).
      - Подождать duration секунд с помощью `WaitForSeconds`.
      - Вернуть скорость к исходному значению.
      - Отключить визуальный эффект (вернуть цвет/масштаб).
  - Если игрок подбирает второе усиление во время активного — старый эффект должен корректно завершиться, а новый начаться (сброс таймера). (Подсказка: используйте `StopCoroutine` перед запуском новой)

### 3. Дополнительно (необязательно, но приветствуется):
  - Добавьте мерцание объекта усиления (мигающий свет или изменение альфа-канала) с помощью второй корутины.
  - При активации усиления проигрывайте звук (через `AudioSource.PlayOneShot`).

---

## 🧪 Ожидаемый результат
- Игрок бегает по сцене.
- Подбирает объект-усиление.
- Скорость увеличивается на 5 секунд.
- Во время усиления персонаж визуально меняется (синий цвет / больше размер).
- Через 5 секунд персонаж возвращается в норму.
