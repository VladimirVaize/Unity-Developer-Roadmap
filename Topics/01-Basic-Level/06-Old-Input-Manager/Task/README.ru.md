# 🧪 Практическое задание: движение, прыжок и стрельба через старый Input Manager

## 📌 Цель задачи
Написать скрипт для простого игрока-куба (Cube), который использует только старый Input Manager:
- Движение вперёд/назад (W/S) и влево/вправо (A/D)
- Прыжок по нажатию пробела
- Стрельба шаром по нажатию левой кнопки мыши (по направлению взгляда камеры)

---

## 🧩 Требования к реализации

### 1. Движение
- Используй `Input.GetAxis("Vertical")` и `Input.GetAxis("Horizontal")`
- Умножай на скорость и `Time.deltaTime`
- Двигай куб через `transform.Translate`

### 2. Прыжок
- Только если игрок на земле (проверка `isGrounded`)
- Используй `Input.GetKeyDown(KeyCode.Space)`
- Добавь `Rigidbody` и применяй силу вверх (`AddForce`)

### 3. Стрельба
- Создавай сферу (префаб) перед игроком
- При клике левой кнопки мыши (`Input.GetMouseButtonDown(0)`)
- Добавь сфере скорость вперёд через `Rigidbody.velocity`

### 4. Бонус (необязательно)
- Добавь бег по удержанию `LeftShift` через `Input.GetKey`

---

## 📁 Структура (пример)
```plaintext
PlayerController.cs  (главный скрипт)
Bullet.prefab        (сфера с коллайдером + Rigidbody)
Ground               (плоскость с коллайдером)
```

---

## ✅ Критерии успеха
- Куб движется плавно и не проваливается сквозь землю
- Прыжок работает только на земле
- При клике мыши вылетает шар вперёд
- Нет ошибок в консоли
