# 🎯 Задача: «Магический пол и подвижная платформа»

Создайте сцену, в которой игрок (сфера) управляется с клавиатуры и взаимодействует с тремя разными зонами, используя коллайдеры, триггеры и физические материалы.

---

## 📌 Требования:
### 1. Обычный пол (Collision + Physic Material)
  - Пол (плоскость) имеет обычный коллайдер (`Is Trigger = false`).
  - Назначьте физический материал «Резина» (высокое трение: `Dynamic Friction = 0.8`, `Bounciness = 0.2`).
  - Игрок должен ходить по полу и не скользить.

### 2. Ледяная зона (Collision + Physic Material)
  - Создайте отдельную платформу с синим цветом.
  - Добавьте физический материал «Лёд» (`Dynamic Friction = 0.05`, `Bounciness = 0.1`).
  - Когда игрок заходит на эту зону, он должен сильно скользить при движении.

### 3. Триггер-ловушка (Trigger)
  - Создайте невидимую зону (куб с `Is Trigger = true`).
  - При входе в зону (событие `OnTriggerEnter`) в консоли должно появляться сообщение: «Осторожно, ловушка!».
  - Игрок проходит сквозь зону без физического отталкивания.

### 4. Подвижная платформа-батут (Collision + Bounciness)
  - Создайте платформу, которая двигается вверх-вниз (анимация или простой скрипт).
  - Назначьте физический материал «Батут» (`Bounciness = 0.9`, `Dynamic Friction = 0.3`).
  - При столкновении с платформой игрок должен высоко подпрыгивать.

### 🧩 Дополнительно (бонус):
  - При столкновении с «ледяной зоной» меняйте цвет игрока на голубой (через скрипт).
  - Сделайте так, чтобы игрок не мог выйти за границы сцены (поставьте стены с коллайдерами).
