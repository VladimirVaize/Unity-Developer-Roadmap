# 🧪 Практическое задание: «Настройки и рекорды с PlayerPrefs»

## 🎯 Цель задания
Создать небольшое игровое приложение в Unity, которое использует `PlayerPrefs` для:
  - Сохранения лучшего рекорда (High Score)
  - Сохранения настроек громкости музыки (Music Volume)
  - Сохранения имени последнего игрока (Last Player Name)

Все данные должны сохраняться между запусками игры.

---

## 📝 Описание задачи
Вам нужно реализовать сцену с UI-элементами:

1. Текст-счётчик очков (Score) — увеличивается на 10 при клике на кнопку `Click!`.
2. Текст лучшего рекорда (High Score) — обновляется, если текущий счёт превышает сохранённый рекорд.
3. Кнопка `Click!` — добавляет 10 очков к текущему счёту.
4. Кнопка `Reset Score` — сбрасывает текущий счёт до 0 (но не трогает рекорд).
5. Слайдер (Slider) для громкости музыки — диапазон 0..1, значение должно сохраняться и применяться к `AudioListener.volume`.
6. Поле ввода (InputField) для имени игрока — сохраняется автоматически при потере фокуса или нажатии Enter.
7. Кнопка `Clear All Data` — вызывает `PlayerPrefs.DeleteAll()` и сбрасывает все UI-элементы в начальное состояние.

---

## ✅ Требования к реализации
- При первом запуске рекорд = 0, громкость = 0.5, имя = `"Player"`.
- При достижении нового рекорда должно появляться сообщение в консоли или на экране.
- Использовать `PlayerPrefs.Save()` после каждого изменения данных.
- В методе `Start()` загружать все сохранённые значения и применять их к UI.
