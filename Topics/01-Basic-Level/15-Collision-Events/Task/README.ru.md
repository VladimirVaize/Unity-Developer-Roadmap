# 🎯 Практическое задание: «Собиратор с препятствиями»

## 📌 Описание задачи

Вам нужно создать простую 3D-сцену в Unity, где игрок управляет сферой (Player), собирает монеты (Coins) и избегает или получает урон от врагов (Enemies). 
Задача закрепляет использование **OnCollisionEnter** и **OnTriggerEnter**, их различия и ограничения.

---

## 🧱 Что нужно сделать

### 1. Настройка сцены
- Создайте плоскость (`Plane`) — пол.
- Добавьте сферу (`Sphere`) — это игрок. Назначьте ей:
  - `Rigidbody` (не кинематический, `isKinematic = false`)
  - `Collider` (оставьте `IsTrigger = false`)
  - Скрипт `PlayerController` для движения по стрелкам/WASD (или используйте стандартный `MoveTowards`).
- Добавьте 5–10 кубов (`Cube`) с тегом `"Coin"`, зелёного цвета. У них должен быть:
  - `Collider` с IsTrigger = true
  - Пустой скрипт (или просто тег).
- Добавьте 3 красных куба с тегом `"Enemy"`. У них:
  - `Collider` (IsTrigger = false)
  - `Rigidbody` (кинематический или нет — на ваше усмотрение, но для столкновения нужен хотя бы один Rigidbody на сцене).
 
### 2. Скрипт игрока (`PlayerController` + события)
Напишите скрипт <a href="../Solution/PlayerCollisionHandler.cs"><code>PlayerCollisionHandler</code></a>, который содержит:
- Хранит значения `score` и `health`
- `OnTriggerEnter(Collider other)` - Столкновение с монетой
  - Проверка через `CompareTag("Coin")`
  - Увеличивает значение `score`
  - Уничтожает монету (`Destroy`)
  - Выводит сообщение `"Монета собрана! Очки: " + score`
- `OnCollisionEnter(Collision collision)` - Столкновение с врагом
  - Проверка через `gameObject.CompareTag("Enemy")`
  - Уменьшает значение `health`
  -  Выводит сообщение `"Урон! Здоровье: " + health`
  -  При `health <= 0` выводит сообщение `"Игра окончена!"`

### 3. Дополнительные требования

- При столкновении с врагом игрок должен отскакивать (используйте `Rigidbody.AddForce`).
- При сборе монеты проигрывайте простой звук (опционально).
- Добавьте стену с триггерной зоной, которая при входе в неё телепортирует игрока в другую точку (используйте `OnTriggerEnter` и `Transform.position`).

## 🧠 Вопросы для самопроверки
- Почему для сбора монет использован `OnTriggerEnter`, а не `OnCollisionEnter`?
- Что произойдёт, если у монеты убрать `Rigidbody`? А если сделать коллайдер не триггером?
- Почему при столкновении с врагом используется `OnCollisionEnter`?

---

## 🏆 Цель
Написать рабочий код, настроить сцену и убедиться, что:
- Монеты исчезают при касании и увеличивают счёт.
- Враги уменьшают здоровье и отталкивают игрока.
- Триггерная зона телепортирует игрока.
