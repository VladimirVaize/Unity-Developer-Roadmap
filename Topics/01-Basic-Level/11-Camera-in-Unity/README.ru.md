# 🎥 Камера в Unity: Проекции, Clear Flags, Culling Mask и Render Texture

> [!NOTE]
> Камера — это «глаза» вашей игры. Она определяет, что и как видит игрок.
> В этом материале разберём четыре ключевых настройки: типы проекций,
> **Clear Flags**, **Culling Mask** и рендеринг в текстуру (**Render Texture**).

---

## 🧭 1. Типы проекций: Perspective vs Orthographic

### 🎭 **Perspective (Перспективная)** – имитирует реальный мир. Объекты выглядят меньше на расстоянии. Используется в 3D-играх (шутеры, RPG, гонки).

### 📐 **Orthographic (Ортографическая)** – нет перспективного искажения. Все объекты отображаются в одинаковом масштабе независимо от расстояния. Идеально для 2D-игр, UI, стратегий (вид сверху).

### ⚙️ Как использовать:
Выберите камеру → в **Inspector** → компонент **Camera** → свойство **Projection** → выберите `Perspective` или `Orthographic`. 
Для ортографической камеры также настройте Size (размер видимой области по вертикали).

### 📌 Пример:
В 3D-платформере используйте `Perspective` для глубины. В шахматной игре сверху — `Orthographic`, чтобы фигуры не искажались по краям.

## 🧹 2. Clear Flags

Определяет, чем заполнять пустые области экрана после отрисовки каждого кадра. Варианты:
- **Skybox** 🌌 – небо (по умолчанию). Пустые места заполняются изображением неба.
- **Solid Color** 🎨 – заливка одним цветом (например, чёрным).
- **Depth Only** 📏 – не очищает цвет, только буфер глубины. Используется для нескольких камер (например, камера интерфейса).
- **Don’t Clear** 🚫 – ничего не очищает (редко, для спецэффектов).

### ⚙️ Как использовать:
Камера → Inspector → Camera → Clear Flags → выберите нужный вариант.

### 📌 Пример:
Главная камера — `Skybox`. Вторая камера для мини-карты — `Depth Only`, чтобы рисовать поверх мира, не стирая небо.

## 🎭 3. Culling Mask (Маска отсечения)
Определяет, какие слои (Layers) объектов будет видеть камера. Позволяет скрыть от камеры определённые объекты (например, интерфейс от основной камеры).

### ⚙️ Как использовать:
1. Назначьте объектам слои (Layers) в инспекторе (например, `UI`, `Player`, `Environment`).
2. Выберите камеру → **Culling Mask** → снимите галочки со слоёв, которые не должна видеть камера.

### 📌 Пример:
Камера игрока видит только слой `Environment`. Камера интерфейса видит только слой `UI`. 
Так элементы интерфейса не будут мешать основному изображению, но будут отображаться поверх.

## 🖼️ 4. Рендеринг в текстуру (Render Texture)
Позволяет выводить изображение с камеры на текстуру, которую можно использовать в игре (например, для экрана монитора, зеркала, камеры наблюдения, мини-карты).

### ⚙️ Как использовать:

1. Правой кнопкой в окне **Project** → `Create` → `Render Texture`. Назовите её, например, `MirrorTexture`.
2. Выберите камеру → в компоненте **Camera** → поле **Target Texture** → перетащите туда `MirrorTexture`.
3. Создайте материал или UI Image, который будет отображать эту текстуру:
   - Для 3D-объекта: перетащите `MirrorTexture` на свойство `Main Texture` материала.
   - Для UI: добавьте `Raw Image` и в поле `Texture` укажите `MirrorTexture`.
  
### 📌 Пример:
Вы хотите сделать экран камеры видеонаблюдения в игре. 
Создайте отдельную камеру, направленную на дверь. 
Её `Target Texture` = `CCTVTexture`. Поместите эту текстуру на плоскость (монитор) в сцене. 
Игрок увидит на мониторе live-изображение двери.
