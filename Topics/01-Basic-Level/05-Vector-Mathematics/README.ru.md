# 📐 Векторная математика в Unity: Основы для игровой логики

Векторы — это фундамент перемещения, направления, расстояний и множества вычислений в игровом движке. 
В Unity чаще всего используются **Vector2** (для 2D) и **Vector3** (для 3D), а также ключевые операции: **Distance**, **Lerp**, **Dot Product** и **Cross Product**. 
Понимание этих инструментов превращает хаотичное движение в управляемую и предсказуемую игровую механику.

---

## 1. Vector2 и Vector3 📦

### Что это:
Структуры, хранящие координаты в пространстве.
- `Vector2` → `(x, y)` — для UI, 2D игр, экранных координат.
- `Vector3` → `(x, y, z)` — для 3D мира, физики, Transform.position.

### Где использовать:
- Позиция объекта: `transform.position = new Vector3(5, 0, 10);`
- Направление движения: `Vector3 direction = target - transform.position;`
- Масштаб: `transform.localScale = new Vector3(2, 2, 2);`

### Полезные константы:
- `Vector3.zero` → (0,0,0)
- `Vector3.one` → (1,1,1)
- `Vector3.up` → (0,1,0)
- `Vector3.forward` → (0,0,1)
- `Vector3.right` → (1,0,0)

### Пример:
Врага нужно сместить вверх на 2 единицы:
```scharp
transform.position += Vector3.up * 2;
```

## 2. Distance 📏

### Что это:
Расстояние между двумя точками в пространстве. Возвращает `float`.

### Формула (внутри Unity):
`Vector3.Distance(A, B) = длина вектора (B - A)`

### Где использовать:
- Проверка, вошёл ли игрок в радиус атаки врага.
- Активация диалога при приближении к NPC.
- Уничтожение пули, пролетевшей максимальную дистанцию.

### Пример:
Враг атакует, если игрок ближе 5 метров:

```csharp
if (Vector3.Distance(transform.position, player.position) < 5f)
{
    Attack();
}
```

## 3. Lerp (Линейная интерполяция) 🔄

### Что это:
Плавный переход от одного значения к другому с течением времени.<br>
`Vector3.Lerp(start, end, t)` где `t` от 0 до 1.

- `t = 0` → `start`
- `t = 1` → `end`
- `t = 0.5` → ровно посередине

### Где использовать:
- Плавное движение камеры за игроком (слежение с отставанием).
- Плавное изменение цвета, размера, позиции.
- Анимация перемещения предмета по дуге.

### Пример:
Плавно переместить объект из точки A в точку B за 2 секунды:

```csharp
float t = Time.time / 2f; // от 0 до 1 за 2 секунды
transform.position = Vector3.Lerp(startPoint, endPoint, t);
```

Совет: Для постоянной скорости используйте `MoveTowards`, а для плавного затухания — `Lerp` с `Time.deltaTime`:

```csharp
transform.position = Vector3.Lerp(transform.position, target, Time.deltaTime * smoothSpeed);
```

## 4. Dot Product (Скалярное произведение) ⚫

### Что это:
Мера того, насколько два направления сонаправлены.<br>
Возвращает `float` от -1 до 1:
- `1` → смотрят строго в одном направлении.
- `0` → перпендикулярны.
- `-1` → строго противоположны.

Формула: `dot = Vector3.Dot(a.normalized, b.normalized)`

### Где использовать:
- Определить, смотрит ли враг на игрока (значение > 0.7).
- Проверка, находится ли объект спереди или сзади.
- Расчёт угла между двумя векторами: `angle = Mathf.Acos(dot) * Mathf.Rad2Deg`.
- Системы освещения (чем больше dot с нормалью поверхности, тем ярче).

### Пример:
Проверка, видит ли враг игрока спереди (не обязательно прямой линией, а именно по направлению взгляда):

```csharp
Vector3 toPlayer = (player.position - transform.position).normalized;
float dot = Vector3.Dot(transform.forward, toPlayer);
if (dot > 0.5f)
{
    Debug.Log("Игрок впереди и в поле зрения!");
}
```

## 5. Cross Product (Векторное произведение) ✖

### Что это:
Вектор, перпендикулярный двум исходным векторам. В 3D позволяет получить направление "влево/вправо" относительно направления взгляда.

### Свойства:
- `Vector3.Cross(a, b)` → результат перпендикулярен и a, и b.
- Длина результата = площадь параллелограмма, построенного на a и b.
- Направление определяется правилом правой руки.

### Где использовать:
- Определение, поворачивать ли врага налево или направо, чтобы смотреть на цель.
- Расчёт оси вращения.
- Построение системы координат (например, кастомная камера).

### Пример:
Узнать, игрок слева или справа от врага (для поведения "обходить слева"):

```csharp
Vector3 toPlayer = player.position - transform.position;
Vector3 cross = Vector3.Cross(transform.forward, toPlayer);
if (cross.y > 0)
    Debug.Log("Игрок справа");
else if (cross.y < 0)
    Debug.Log("Игрок слева");
```

## 🧠 Шпаргалка для игровой логики

| Задача | Инструмент | Пример использования |
|----------------|-------------|-------------------------|
| Позиция объекта | Vector3 | `transform.position` |
| Расстояние до цели | Distance | `if(Distance < 5) Attack()` |
| Плавное движение | Lerp | `pos = Lerp(pos, target, dt)` |
| Смотрит ли на меня? | Dot > 0.7 | `if(Dot(forward, toTarget) > 0.7f)` |
| Слева или справа? | Cross.y | `float side = Cross(forward, toTarget).y` |
